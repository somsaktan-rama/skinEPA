# 🔧 คู่มือการใช้งานระบบ EPA Skin Tracker
**สำหรับผู้ดูแลระบบ (Superstaff)**

สิทธิ์ Superstaff ถูกออกแบบมาเพื่อให้ผู้ดูแลระบบสามารถจัดการฐานข้อมูลบุคลากร และแก้ไขปัญหาการเข้าใช้งาน (เช่น การผูกบัญชี LINE) ให้กับผู้ใช้งานท่านอื่นผ่านหน้าเว็บไซต์ได้โดยตรง โดยไม่จำเป็นต้องเข้าไปแก้ไขใน Google Sheets หลังบ้าน

---

## 1. 🚪 การเข้าสู่หน้าจอจัดการข้อมูล (Superstaff Menu)
1. ล็อกอินเข้าสู่ระบบด้วยบัญชีที่มีสิทธิ์กำกับไว้ว่า `superstaff`
2. ที่หน้าเมนูหลัก ให้เลื่อนลงมาด้านล่างสุด จะพบปุ่มกรอบสีแดงเขียนว่า **"🔧 จัดการข้อมูลบุคลากร"** ให้คลิกเพื่อเข้าสู่โหมดผู้ดูแลระบบ

---

## 2. 🔍 การค้นหาและแก้ไขข้อมูลบุคลากร
1. ในหน้า Superstaff ให้กรอก **รหัส MDID** (เฉพาะตัวเลข 4-10 หลัก) ของบุคคลที่ต้องการแก้ไขข้อมูลลงในช่องค้นหา แล้วกดปุ่ม **"ดึงข้อมูล"**
2. ระบบจะแสดงข้อมูลของบุคลากรท่านนั้นขึ้นมา พร้อมกับภาพโปรไฟล์และเวลาที่ล็อกอินเข้าใช้งานล่าสุด (Last Login)
3. ท่านสามารถคลิกแก้ไขข้อมูลในฟิลด์ต่างๆ ได้ทันที เช่น:
   * ข้อมูลทั่วไป: ชื่อ (Name), สถาบัน (Institute), บทบาท (Role)
   * ข้อมูลการศึกษา: ปีที่รับเข้าศึกษา (Admission Year), ข้อมูลชั้นปี R2/R3/R4, สถานะการศึกษา (Status)
   * ข้อมูลติดต่อ: อีเมล (Email), เบอร์โทรศัพท์ (Mobile)
4. เมื่อทำการแก้ไขข้อมูลเสร็จเรียบร้อยแล้ว ให้เลื่อนลงมากดปุ่ม **"บันทึกข้อมูล"** สีเขียว

---

## 3. 🎯 การจัดการสิทธิ์พิเศษสำหรับอาจารย์ (Instructor Targets)
ในหน้าต่างแก้ไขข้อมูลบุคลากร จะมี 2 ฟิลด์สำคัญที่ใช้สำหรับกำหนดสิทธิ์และขอบเขตการทำงานของอาจารย์แพทย์แต่ละท่าน:

* **Eval Target (เป้าหมายการประเมิน):** ใช้ระบุ MDID ของลูกศิษย์หรือกลุ่มที่อาจารย์ท่านนี้ต้องประเมินเป็นพิเศษ (หากมีหลายรหัสให้คั่นด้วยเครื่องหมายลูกน้ำ `,`)
* **View Target (สิทธิ์การมองเห็นผล 360 องศา):** ใช้กำหนดว่าอาจารย์ท่านนี้สามารถเข้าไปดูผลประเมิน 360 องศาของลูกศิษย์คนใดได้บ้าง หากปล่อยช่องนี้ว่างไว้ ระบบจะทำการซ่อนปุ่มเมนู "ดูผลประเมิน 360 (ลูกศิษย์)" ออกจากหน้าจอหลักของอาจารย์ท่านนั้นโดยอัตโนมัติ
* ** Advisor MDID (อาจารย์ที่ปรึกษา): ใช้สำหรับระบุรหัส MDID ของอาจารย์ที่เป็นที่ปรึกษาประจำตัวให้กับ Resident หรือ Fellow ท่านนั้น (เป็นข้อมูลความสัมพันธ์พื้นฐาน ไม่มีผลกับการซ่อน/แสดงเมนู)
---

## 4. 🔓 การปลดล็อก/รีเซ็ตการผูกบัญชี LINE (Unbind LINE)
หากมีบุคลากรแจ้งปัญหาว่า *"ล็อกอินไม่ได้ เนื่องจากบัญชีถูกผูกกับเครื่องอื่นไปแล้ว"* (เช่น กรณีเปลี่ยนโทรศัพท์มือถือ หรือเผลอไปล็อกอินผิดเครื่อง) Superstaff สามารถรีเซ็ตระบบให้ได้ทันทีตามขั้นตอนต่อไปนี้:

1. ค้นหา MDID ของผู้ใช้งานที่พบปัญหา
2. เลื่อนลงมาหาช่องที่ชื่อว่า **"LINE UID (ลบออกเพื่อรีเซ็ตเครื่องใหม่ได้)"**
3. **ทำการลบข้อความ (รหัสตัวอักษรภาษาอังกฤษยาวๆ) ในช่องนั้นทิ้งทั้งหมด** ให้กลายเป็นช่องว่างเปล่า
4. เลื่อนลงมากดปุ่ม **"บันทึกข้อมูล"**
5. แจ้งให้บุคลากรท่านนั้นกลับไปเปิดแอปพลิเคชันและทำการเข้าสู่ระบบ (ล็อกอิน) ใหม่อีกครั้งที่อุปกรณ์เครื่องใหม่ ระบบจะทำการผูกบัญชีเข้ากับ LINE เครื่องใหม่ให้อัตโนมัติครับ

> **💡 เกร็ดความรู้:** ช่อง "LINE Display Name" จะดึงชื่อโปรไฟล์ LINE ของผู้ใช้งานมาแสดงแบบอ่านได้อย่างเดียว (Read-only) เพื่อให้ Superstaff ตรวจสอบได้ว่ารหัส MDID นี้ถูกผูกอยู่กับ LINE ของใครครับ
