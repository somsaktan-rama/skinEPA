📘 คู่มือการใช้งานระบบ skinEPA V 2.30 สำหรับ Resident / Fellow

1. การเข้าสู่ระบบ (Authentication)

ระบบใช้กลไกการยืนยันตัวตนแบบ MdID & LINE UID Binding

  • Auto-Login: เมื่อเปิดแอปผ่าน LINE ระบบจะดึงรหัสเข้าสู่เมนูหลักตามสิทธิ์ทันที
  • การเข้าใช้งานครั้งแรก: ระบบจะบังคับให้เข้าสู่ระบบด้วย RamaID เพื่อยืนยันตัวตนก่อนจับคู่กับ LINE UID

2. ฟีเจอร์หลักสำหรับผู้เรียน

📍 การลงชื่อเข้าร่วมกิจกรรมวิชาการ

  • On-site: กดปุ่มสีเขียว ลงชื่อ On-site ระบบจะส่งพิกัด GPS เพื่อตรวจสอบระยะรัศมีหน้าห้องเรียนอัตโนมัติ
  • Hybrid / Online: ต้องกรอก PIN 4 หลัก ที่อาจารย์ประกาศในห้องเรียนเพื่อยืนยัน

📝 การสร้างฟอร์ม EPA

  • เลือกรายการหัตถการและอาจารย์ผู้ประเมิน (ระบบจะซ่อนชื่ออาจารย์ที่ประเมินครบโควต้าอัตโนมัติ)
  • ข้อควรระวัง: ผู้เรียนจำเป็นต้องกรอกรายละเอียดเคส (เช่น ข้อมูลโรค, เทคนิค) ในช่อง Note ห้ามเว้นว่างเด็ดขาด

📊 การเข้าดูผล 360° (6 มิติ)

แสดงผลกราฟเรดาร์ 6 มิติ (Teamwork, Communication, Punctuality, EQ, Leadership, Respect)

🚨 เงื่อนไขความปลอดภัย (Advisor Gatekeeper):
ผู้เรียนจะมองเห็นคะแนนและข้อเสนอแนะได้ ก็ต่อเมื่ออาจารย์ที่ปรึกษา (Advisor) กดยืนยันอนุมัติ (Allow) ข้อความนั้นผ่านระบบหลังบ้านแล้วเท่านั้น!

3. วิธีติดตั้งแอปบนหน้าจอมือถือ

🍎 สำหรับ iOS (iPhone)

  1. กดคัดลอกลิงก์จากแอป LINE
  2. นำลิงก์ไปเปิดใน Safari
  3. แตะไอคอนแชร์ (Share) ด้านล่าง
  4. เลือก เพิ่มไปยังหน้าจอโฮม (Add to Home Screen)

🤖 สำหรับ Android

  1. เลือก เปิดในเบราว์เซอร์เริ่มต้น (Chrome)
  2. แตะเมนู 3 จุด มุมขวาบนใน Chrome
  3. เลือก เพิ่มลงในหน้าจอหลัก (Add to Home screen)
