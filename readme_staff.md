# คู่มือการใช้งานระบบ skinEPA เวอร์ชั่น 2.30 (สำหรับ Staff / Admin)

ระบบ **skinEPA Tracker V 2.30** สำหรับเจ้าหน้าที่ (Staff / Coordinator) ออกแบบมาเพื่อให้สามารถบริหารจัดการข้อมูลกิจกรรมวิชาการประจำวัน ควบคุมการเช็คอินหน้าห้องเรียน ตลอดจนส่งประกาศเตือนผ่านเทคโนโลยี LINE API

---

## ⚙️ 1. ฟีเจอร์หลักสำหรับผู้ประสานงาน

### 1.1 การบริหารจัดการตารางกิจกรรมวิชาการ
* **การเปิดกิจกรรมใหม่แบบรายครั้ง (Single Session):** เข้าเมนู *"📌 เปิดกิจกรรมใหม่"* เพื่อบันทึกข้อมูลรหัสกิจกรรม, หัวข้อเรื่อง, ผู้สอน, และวันเวลา
* **ระบบจัดการแม่แบบรายสัปดาห์ (Weekly Batch Template):** สร้างแม่แบบล่วงหน้า และเข้าเมนู *"📅 ระบบแม่แบบรายสัปดาห์"* กดปุ่ม **"ยืนยันสร้างกิจกรรมที่เลือก"** ระบบจะสั่งเปิดตารางกิจกรรมทั้งสัปดาห์ (สูงสุด 10 คลาส) ในคลิกเดียว

### 1.2 การควบคุมห้องเรียนและการเข้าเรียนในพื้นที่ (Real-time Attendance Control)
เข้าเมนู *"⚙️ จัดการกิจกรรม"* ระบบจะแสดง:
1. **รหัส Check-in (PIN):** แสดงรหัสตัวเลขสุ่ม 4 หลักประจำคลาส
2. **หน้าจอตรวจสอบรายชื่อผู้เข้าร่วม (Live Attendee Tracker):** แสดงสัญลักษณ์สีพิกัด GPS:
   * *สีม่วง (Instructor):* พิกัดศูนย์กลางอาจารย์ผู้คุมคลาส
   * *สีเขียว (In Range):* ผู้เรียนอยู่ในรัศมีระยะปลอดภัย
   * *สีแดง (Out of Range):* อยู่นอกพื้นที่คณะวิชา (ระบุระยะห่างจริง)
   * *สีฟ้า (Wait Instructor):* ผู้เรียนลงชื่อแล้ว แต่รออาจารย์กดลงชื่อเปิดคลาส
   * *สีเทา:* ไม่ระบุพิกัด (ออนไลน์)
3. **ปุ่มสิทธิ์คำสั่งพิเศษหน้าห้องเรียน:**
   * **ปุ่มนำออก:** ถอนชื่อผู้ที่ทุจริตออก
   * **ปุ่มแก้ไข (✏️ แก้ไข):** อัปเดตเนื้อหาหรือขยายเวลา
   * **ปุ่มลบ (🗑️ ลบ):** จะแสดงก็ต่อเมื่อ **"มียอดผู้เข้าร่วมเรียนเป็น 0 คนเท่านั้น"**
   * **ปุ่มปิดกิจกรรม:** กดปิดรับการลงชื่อถาวร

### 1.3 ระบบส่งประกาศแจ้งเตือนความปลอดภัยสูง (Targeted LINE OA Broadcast)
ระบบตรวจสอบแบบ 2 จังหวะ (Two-Step Verification) ป้องกันการส่งข้อความขยะ:
1. **จังหวะที่ 1 (Count-Only API Check):** เลือกกลุ่มเป้าหมาย (เช่น Resident 2, Fellow) แล้วกดส่ง ระบบจะแสดงป๊อปอัพชี้แจงยอดบัญชีที่ผูกระบบ LINE สำเร็จก่อน
2. **จังหวะที่ 2 (Multicast Send):** เมื่อกดยืนยัน ระบบจะส่งข้อความรูปแบบ `📢 [ประกาศจาก: ชื่อเจ้าหน้าที่]` ยิงตรงเข้าสู่ LINE ของผู้ใช้ผ่าน Multicast API ทันที

### 1.4 แดชบอร์ดสรุปความก้าวหน้าและการติดตามผล (Analytics)
* **Dashboard ความก้าวหน้า EPA:** แสดงป้ายกำกับสถานะ **"🚩 เสี่ยงล่าช้า (At Risk)"** สีแดง หากเก็บเคสช้ากว่าเกณฑ์ เจ้าหน้าที่สามารถกดปุ่ม **"📥 Export Excel / Pivot"** เพื่อดึงข้อมูลดิบของตารางแบบ Real-time ออกมาเป็นไฟล์ตารางสำหรับทำรายงาน
* **ระบบติดตามยอดประเมิน 360:** หากแพทย์คนใดได้รับการประเมินยอดรวม **น้อยกว่า 5 ฉบับ ระบบจะแสดงแถบไฮไลท์สีแดงแจ๋** ให้เจ้าหน้าที่เร่งติดตามประสานงาน

## 2. 📅 การนำเข้าและซิงค์ตารางกิจกรรม (Google Calendar Sync)
ท่านสามารถเชื่อมต่อตารางกิจกรรมของสถาบันเข้าสู่ปฏิทินส่วนตัวบนสมาร์ทโฟนได้อัตโนมัติ ระบบจะอัปเดตวัน-เวลาให้เองหากมีการเปลี่ยนแปลง

1. ไปที่ปุ่ม **"📖 คู่มือการใช้งานระบบ"** ในหน้าเมนูหลัก
2. ในหน้าต่างคู่มือ จะมีกล่องข้อความสำหรับซิงค์ปฏิทิน ให้กดปุ่ม **"➕ กดเพื่อติดตั้ง/ติดตามปฏิทิน"**
3. **วิธีเปิดแสดงผลบนสมาร์ทโฟน:**
   * **สำหรับ Android (แอป Google Calendar):** เปิดแอป > ไปที่เมนู 3 ขีด (มุมซ้ายบน) > การตั้งค่า > กดชื่อปฏิทินสถาบัน > เปิดสวิตช์ "ซิงค์ (Sync)"
   * **สำหรับ iPhone (แอปปฏิทินติดเครื่อง):** หลังจากกดปุ่มติดตั้งแล้ว ให้เข้าเบราว์เซอร์ไปที่ `calendar.google.com/calendar/syncselect` แล้วติ๊กถูกหน้าชื่อปฏิทินสถาบันเพื่อเปิดระบบดาวน์โหลดข้อมูลครับ
