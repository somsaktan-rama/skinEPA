# ⚙️ คู่มือการใช้งานระบบ EPA Skin Tracker (อัปเดตระบบ Hybrid Learning)
**สำหรับ พยาบาลวิชาชีพ, นักวิทยาศาสตร์, Clerk และ เจ้าหน้าที่สายสนับสนุน (Staff / Admin)**

ระบบ EPA Skin Tracker ถูกออกแบบมาเพื่อช่วยให้เจ้าหน้าที่และ Admin สามารถจัดการกิจกรรมวิชาการ และติดตามความก้าวหน้าของการประเมินของแพทย์ประจำบ้านได้อย่างมีประสิทธิภาพ

# คู่มือการใช้งานระบบ skinEPA เวอร์ชั่น 2.20 (Comprehensive User Manual)

ระบบ **skinEPA Tracker V 2.20** เป็นแพลตฟอร์มดิจิทัลที่พัฒนาขึ้นบนโครงสร้างเทคโนโลยี **LINE Front-end Framework (LIFF)** เชื่อมต่อฐานข้อมูลประสิทธิภาพสูงแบบ Real-time ร่วมกับ **Google Apps Script (GAS)** และ **Google Sheets** มีวัตถุประสงค์หลักเพื่อบันทึก ประเมิน และติดตามความก้าวหน้าทางการศึกษาของแพทย์ประจำบ้าน (Resident) และแพทย์ประจำบ้านต่อยอด (Fellow) ทั้งในส่วนของหัตถการหลัก (EPA), การเข้าร่วมกิจกรรมวิชาการ (Academic Attendance) และการประเมินผลรอบทิศทาง 360 องศา โดยเวอร์ชั่น 2.20 นี้ได้พัฒนาโครงสร้างความปลอดภัย ระบบคัดกรองข้อมูลโดยอาจารย์ที่ปรึกษา และระบบสื่อสารความเร็วสูงผ่าน LINE API อย่างสมบูรณ์

## 🔐 โครงสร้างการเข้าสู่ระบบและการยืนยันตัวตน (Authentication Flow)
ระบบใช้กลไกการยืนยันตัวตนแบบ **LINE UID Binding Cross-Reference** ควบคุมป๊อปอัปแจ้งเตือนทั้งหมดด้วยโปรแกรม **SweetAlert2 (Swal)** ซึ่งมีลอจิกการทำงานดังนี้:
1. **Auto-Login:** เมื่อเปิดแอปพลิเคชันผ่าน LINE ระบบจะดึงรหัสรหัสลับประจำตัว `line_uid` ของแอปพลิเคชัน LINE อัตโนมัติ หากรหัสนั้นเคยผูกไว้ในฐานข้อมูลชีต `staff` แล้ว ระบบจะอนุญาตให้ข้ามหน้าจอลงทะเบียนและเข้าสู่เมนูหลักตามสิทธิ์ (Role-Based Access Control) ทันทีภายใน 1 วินาที
2. **First-Time Login:** หากเป็นผู้ใช้งานใหม่หรือยังไม่เคยผูกบัญชี LINE ระบบจะเปิดหน้าจอให้กรอกรหัส **MDID (รหัส ว. หรือเบอร์มือถือตัวเลข 4-10 หลัก)** * หากรหัส MDID ตรงกับฐานข้อมูลและยังไม่มีใครผูกบัญชี ระบบจะทำการล็อกรหัส `line_uid` และชื่อ `line_displayname` เข้ากับแถวข้อมูลนั้นทันที
   * หากรหัส MDID นั้นเคยถูกผู้อื่นผูกไปแล้ว ระบบจะแจ้งเตือนความปลอดภัยข้อความสีแดงเพื่อป้องกันสิทธิ์ พร้อมเสนอลิงก์ให้กดแจ้งความประสงค์ขอรีเซ็ตสิทธิ์การผูกบัญชีผ่านอีเมลที่ลงทะเบียนไว้

## 2. 📅 การนำเข้าและซิงค์ตารางกิจกรรม (Google Calendar Sync)
ท่านสามารถเชื่อมต่อตารางกิจกรรมของสถาบันเข้าสู่ปฏิทินส่วนตัวบนสมาร์ทโฟนได้อัตโนมัติ ระบบจะอัปเดตวัน-เวลาให้เองหากมีการเปลี่ยนแปลง

1. ไปที่ปุ่ม **"📖 คู่มือการใช้งานระบบ"** ในหน้าเมนูหลัก
2. ในหน้าต่างคู่มือ จะมีกล่องข้อความสำหรับซิงค์ปฏิทิน ให้กดปุ่ม **"➕ กดเพื่อติดตั้ง/ติดตามปฏิทิน"**
3. **วิธีเปิดแสดงผลบนสมาร์ทโฟน:**
   * **สำหรับ Android (แอป Google Calendar):** เปิดแอป > ไปที่เมนู 3 ขีด (มุมซ้ายบน) > การตั้งค่า > กดชื่อปฏิทินสถาบัน > เปิดสวิตช์ "ซิงค์ (Sync)"
   * **สำหรับ iPhone (แอปปฏิทินติดเครื่อง):** หลังจากกดปุ่มติดตั้งแล้ว ให้เข้าเบราว์เซอร์ไปที่ `calendar.google.com/calendar/syncselect` แล้วติ๊กถูกหน้าชื่อปฏิทินสถาบันเพื่อเปิดระบบดาวน์โหลดข้อมูลครับ

## ⚙️ ส่วนที่ 3: คู่มือการใช้งานสำหรับ Staff / พยาบาล / เจ้าหน้าที่ (ผู้ประสานงานหลัก)

สิทธิ์จัดการระบบระดับเจ้าหน้าที่ (Staff / Coordinator) ออกแบบมาเพื่อให้สามารถบริหารจัดการข้อมูลกิจกรรมวิชาการประจำวัน ควบคุมการเช็คอินหน้าห้องเรียน ตลอดจนส่งประกาศเตือนผ่านเทคโนโลยี LINE API

### 1. การบริหารจัดการตารางกิจกรรมวิชาการ (Single & Weekly Batch System)
* **การเปิดกิจกรรมใหม่แบบรายครั้ง (Single Session):** เข้าเมนู *"📌 เปิดกิจกรรมใหม่ (แบบครั้งเดียว)"* เพื่อบันทึกข้อมูลรหัสกิจกรรม, พิมพ์หัวข้อเรื่อง (Topic), เลือกอาจารย์แพทย์ผู้สอนประจำคาบ, บันทึกวันเวลาเริ่มต้น และเลือกระยะเวลาจัดงาน (30-180 นาที) ระบบจะคำนวณเวลาสิ้นสุดให้อัตโนมัติ *พิเศษ:* หากเป็นการเรียนแบบออนไลน์ ให้วางลิงก์ Zoom/Webex ในช่องลิงก์ประชุม ระบบจะเปิดการกรอกแบบใช้รหัส PIN ทันที
* **ระบบจัดการแม่แบบรายสัปดาห์ (Weekly Batch Template):** ออกแบบมาเพื่อลดภาระงานของเจ้าหน้าที่ในการพิมพ์ข้อมูลกิจกรรมซ้ำๆ ทุกสัปดาห์ โดยระบุข้อมูลกิจกรรมหลักประจำวันจันทร์-ศุกร์ พร้อมลิงก์ประชุมออนไลน์ห้องประจำไว้เป็นแม่แบบ เมื่อขึ้นสัปดาห์ใหม่ เจ้าหน้าที่เพียงเข้าเมนู *"📅 ระบบแม่แบบรายสัปดาห์"* ติ๊กเลือกแม่แบบกิจกรรมที่ต้องการเปิดใช้งาน แล้วกดปุ่ม **"ยืนยันสร้างกิจกรรมที่เลือก"** ระบบจะคำนวณวันที่จริงของสัปดาห์ถัดไปตามปฏิทินและสั่งเปิดตารางกิจกรรมพร้อมกันทั้งหมด 10 คลาสภายในคลิกเดียว โดยข้อมูลแม่แบบจะถูกจัดเรียงตามลำดับวันและเวลาเรียนที่ใกล้จะถึงที่สุดให้อัตโนมัติ และมีปุ่มถังขยะ `🗑️` เพื่อสั่งลบแม่แบบที่เลิกใช้งานได้อย่างสะดวก

### 2. การควบคุมห้องเรียนและการเข้าเรียนในพื้นที่ (Real-time Attendance Control)
เมื่อเจ้าหน้าที่กดเลือกรายการคำสั่งควบคุมกิจกรรมวิชาการในเมนู *"⚙️ จัดการกิจกรรม"* ระบบจะแสดงรายละเอียดสถิติห้องเรียนและกลไกความปลอดภัยหน้างาน:
1. **รหัส Check-in (PIN):** แสดงผลรหัสตัวเลขสุ่ม 4 หลักประจำคลาสเด่นชัดบนหน้าต่าง เพื่อให้เจ้าหน้าที่นำไปแสดงบนหน้าจอโปรเจคเตอร์หรือแจ้งให้อาจารย์ประกาศให้เฉพาะผู้ที่นั่งเรียนในห้องทราบ
2. **หน้าจอตรวจสอบรายชื่อผู้เข้าร่วม (Live Attendee Tracker):** แสดงรายชื่อผู้เช็คอินสำเร็จแบบ Real-time พร้อมแสดงสัญลักษณ์สีบ่งชี้คุณภาพความปลอดภัยพิกัดดาวเทียม (GPS Distance Matching) อย่างละเอียด:
   * *สีม่วง (Instructor):* สถานะอาจารย์แพทย์ผู้คุมคลาส (ใช้เป็นพิกัดจุดศูนย์กลางกลางห้องเรียน)
   * *สีเขียว (In Range):* ผู้เรียนเช็คอินอยู่ในรัศมีระยะปลอดภัยที่กำหนดรอบตัวอาจารย์หรือพิกัดคณะ (ระบุระยะห่างเป็นกิโลเมตรชัดเจน)
   * *สีแดง (Out of Range):* ตรวจพบผู้เรียนกดลงชื่ออยู่นอกพื้นที่คณะวิชา (ระบุระยะห่างจริง เช่น ห่างออกไป 15.5 กม. ป้องกันการแอบกดลงชื่อจากที่บ้าน)
   * *สีฟ้า (Wait Instructor):* ผู้เรียนกดลงชื่อในระบบ On-site แล้ว แต่ต้องรอให้อาจารย์แพทย์ประจำคาบกดลงชื่อเข้าคลาสก่อน เพื่อนำพิกัดดาวเทียมมาเทียบระยะห่างความถูกต้อง
   * *สีเทา (ไม่ระบุพิกัด):* กรณีเข้าเรียนผ่านทางออนไลน์ ซูม หรืออุปกรณ์ปิดกั้นสัญญาณดาวเทียม
3. **ปุ่มสิทธิ์คำสั่งพิเศษหน้าห้องเรียน:**
   * **ปุ่มนำออก:** เจ้าหน้าที่สามารถกดปุ่มสีแดงเพื่อถอนชื่อผู้ที่ทุจริตหรือกดรายชื่อผิดออกจากคลาสได้ทันที ยอดสถิติจะคำนวณใหม่แบบสดๆ
   * **ปุ่มแก้ไข (✏️ แก้ไข):** แสดงปุ่มเมื่อสถานะห้องเรียนเป็น `OPEN` เพื่อให้อัปเดตเนื้อหาผู้สอน ลิงก์ซูม หรือขยายเวลาเรียนฉุกเฉินได้ทันทีผ่าน Popup กล่อง SweetAlert2
   * **ปุ่มลบ (🗑️ ลบ):** ระบบจำกัดสิทธิ์ความปลอดภัยสูงสุด จะยอมให้แสดงปุ่มลบกิจกรรมทิ้งก็ต่อเมื่อ **"มียอดผู้เข้าร่วมเรียนเป็น 0 คนเท่านั้น"** หากมีคนกดลงชื่อแล้วจะลบไม่ได้ เพื่อป้องกันการสูญหายของข้อมูลโดยอุบัติเหตุ
   * **ปุ่มปิดกิจกรรม:** จะปรากฏขึ้นโดยอัตโนมัติเมื่อเวลาดำเนินกิจกรรมเดินเลยกำหนดเวลาสิ้นสุดของคลาสวิชาการนั้นๆ แล้ว เพื่อให้เจ้าหน้าที่กดปิดรับการลงชื่อถาวร

### 3. ระบบส่งประกาศแจ้งเตือนความปลอดภัยสูง (Targeted LINE OA Broadcast System)
ฟีเจอร์สื่อสารตรงไปยังไลน์ส่วนตัวของกลุ่มบุคลากรในสถาบันแยกตามประเภทบทบาท โดยระบบออกแบบกลไกป้องกันความผิดพลาดและการส่งข้อความขยะ (Spam) เป็น **ระบบตรวจสอบแบบ 2 จังหวะ (Two-Step Verification)**:
1. **การคัดกรองเป้าหมาย:** เจ้าหน้าที่ติ๊กเลือกกลุ่มบทบาทที่ต้องการประกาศในหน้าจอ ได้แก่ Resident 2, Resident 3, Resident 4, Fellow, Instructor, Nurse หรือกลุ่มพนักงานสายสนับสนุน (Other) จากนั้นพิมพ์ข้อความประกาศในช่องคำสั่ง
2. **จังหวะที่ 1 (Count-Only API Check):** เมื่อกดปุ่มส่ง ระบบจะส่ง Payload พิเศษแบบแอบสำรวจ (`count_only: true`) ไปที่หลังบ้านเพื่อประมวลผลความหนาแน่นของข้อมูล โดยจะแสดงผลป๊อปอัพชี้แจงยอดตัวเลขเปรียบเทียบชัดเจนก่อนส่งจริง:
   * *“กลุ่มเป้าหมายที่ท่านเลือกมีจำนวนทั้งหมด 18 คน บัญชีที่ผูกระบบ LINE OA สำเร็จและพร้อมรับข้อความจำนวน 15 คน ระบบจะส่งข้อความไปยังผู้ที่ผูกไลน์แล้วเท่านั้น”*
3. **จังหวะที่ 2 (Multicast Send):** เมื่อตรวจสอบความถูกต้องและกดยืนยัน ระบบหลังบ้านจะนำข้อความมาประกอบร่างประกอบชื่อผู้ส่งให้อัตโนมัติในรูปแบบ: `📢 [ประกาศจาก: ชื่อเจ้าหน้าที่] 

 [เนื้อหาข้อความประกาศ]` และยิงตรงเข้าสู่สมาร์ทโฟนของผู้ใช้ผ่านระบบ LINE Multicast API ทันที พร้อมเก็บบันทึกข้อความล่าสุดลงในความจำเครื่อง (`localStorage`) เพื่อไม่ให้ข้อความหายเมื่อปิดหน้าจอ

### 4. แดชบอร์ดสรุปความก้าวหน้าและการติดตามผลเชิงบริหาร (Analytics & Administration)
* **Dashboard ความก้าวหน้า EPA:** แสดงรายชื่อผู้เรียนทั้งหมดในสถาบันแยกสัดส่วนระหว่างแพทย์ประจำบ้าน (RES) และแพทย์ต่อยอด (FELLOW) ตรวจสอบและแสดงป้ายกำกับสถานะ **"🚩 เสี่ยงล่าช้า (At Risk)"** สีแดง พร้อมแสดงแถบความสำเร็จโดยรวม หากความเร็วในการเก็บเคสต่ำกว่าเกณฑ์ระยะเวลาปีการศึกษาจริงของคณะ เจ้าหน้าที่สามารถคลิกที่แถบรายชื่อเพื่อกางดูตารางสถิติจำนวนเคสที่ผ่านเกณฑ์ของเด็กคนนั้นแยกตามหัวข้อ EPA เพื่อวินิจฉัยปัญหาการเรียนได้ทันที และมีปุ่ม **"📥 Export Excel"** เพื่อดึงข้อมูลดิบทั้งหมดออกมาเป็นไฟล์ตาราง CSV รองรับการทำงานในโปรแกรม Microsoft Excel
* **ระบบติดตามยอดประเมิน 360 (Admin Tracking):** แสดงรายชื่อแพทย์ประจำบ้านทุกคนพร้อมสรุปจำนวนฟอร์มประเมิน 360 องศาที่ได้รับจริงแยกตามแหล่งที่มา 4 มิติผู้ประเมิน โดยระบบจะทำการกำหนดเกณฑ์ตรวจสอบสากล หากแพทย์คนใดได้รับการประเมินยอดรวม **น้อยกว่า 5 ฉบับ ระบบจะแสดงแถบไฮไลท์สีแดงแจ๋ทันที** เพื่อส่งสัญญาณเตือนให้เจ้าหน้าที่ดำเนินการประสานงานติดตามผู้เกี่ยวข้องให้เร่งส่งฟอร์มก่อนปิดเทอม

## 📱 วิธีติดตั้งแอป EPA Skin ลงบนหน้าจอมือถือ (Home Screen)
เพื่อความสะดวกในการเข้าใช้งานครั้งต่อไปแบบเต็มจอ (Full Screen) โดยไม่ต้องเข้าผ่านแอป LINE ท่านสามารถติดตั้งระบบไว้บนหน้าจอมือถือได้ตามขั้นตอนดังนี้:

### 🍎 สำหรับ iOS (iPhone / iPad)
เนื่องจากระบบความปลอดภัยของ LINE ท่านต้องนำลิงก์ไปเปิดใน Safari ด้วยตนเองดังนี้:
1. เมื่อกดลิงก์เข้ามาแล้ว (ในแอป LINE) ให้แตะที่ไอคอน **3 จุด (⋮)** ที่มุมขวาบนสุด
2. เลือกเมนู **"คัดลอกลิงก์" (Copy Link)** 
3. ออกจากแอป LINE แล้วไปเปิดแอป **Safari** ที่หน้าจอ iPhone ของท่าน
4. วางลิงก์ที่คัดลอกมาลงในช่องค้นหา (Address Bar) ด้านบน แล้วกดเปิดเว็บ
5. เมื่อหน้าเว็บโหลดเสร็จ ให้แตะไอคอน **"แชร์" (Share)** ⍗ (รูปสี่เหลี่ยมมีลูกศรชี้ขึ้น) ที่แถบด้านล่าง
6. เลื่อนหาและเลือกเมนู **"เพิ่มไปยังหน้าจอโฮม" (Add to Home Screen)** 
7. กดปุ่ม **"เพิ่ม" (Add)** มุมขวาบน *(ไอคอนแอปจะไปปรากฏอยู่ที่หน้าจอหลักทันที)*

---

### 🤖 สำหรับ Android
1. เมื่อกดลิงก์เข้ามาแล้ว (ในแอป LINE) ให้แตะที่ไอคอน **3 จุด (⋮)** ที่มุมขวาบนสุด
2. เลือกเมนู **"เปิดในเบราว์เซอร์เริ่มต้น"** หรือ **"Open in Browser"** (หน้าเว็บจะเด้งไปเปิดใน Google Chrome)
3. เมื่ออยู่ใน Chrome ให้แตะไอคอนเมนู **3 จุด (⋮)** ที่มุมขวาบนของหน้าจออีกครั้ง
4. เลือกเมนู **"เพิ่มลงในหน้าจอหลัก" (Add to Home screen)**
5. กดปุ่ม **"เพิ่ม" (Add)** เพื่อยืนยันการติดตั้ง
