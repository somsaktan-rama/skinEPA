# คู่มือการใช้งานระบบ skinEPA เวอร์ชั่น 2.20 (Comprehensive User Manual)

ระบบ **skinEPA Tracker V 2.20** เป็นแพลตฟอร์มดิจิทัลที่พัฒนาขึ้นบนโครงสร้างเทคโนโลยี **LINE Front-end Framework (LIFF)** เชื่อมต่อฐานข้อมูลประสิทธิภาพสูงแบบ Real-time ร่วมกับ **Google Apps Script (GAS)** และ **Google Sheets** มีวัตถุประสงค์หลักเพื่อบันทึก ประเมิน และติดตามความก้าวหน้าทางการศึกษาของแพทย์ประจำบ้าน (Resident) และแพทย์ประจำบ้านต่อยอด (Fellow) ทั้งในส่วนของหัตถการหลัก (EPA), การเข้าร่วมกิจกรรมวิชาการ (Academic Attendance) และการประเมินผลรอบทิศทาง 360 องศา โดยเวอร์ชั่น 2.20 นี้ได้พัฒนาโครงสร้างความปลอดภัย ระบบคัดกรองข้อมูลโดยอาจารย์ที่ปรึกษา และระบบสื่อสารความเร็วสูงผ่าน LINE API อย่างสมบูรณ์

---

## 🔐 โครงสร้างการเข้าสู่ระบบและการยืนยันตัวตน (Authentication Flow)
ระบบใช้กลไกการยืนยันตัวตนแบบ **LINE UID Binding Cross-Reference** ควบคุมป๊อปอัปแจ้งเตือนทั้งหมดด้วยโปรแกรม **SweetAlert2 (Swal)** ซึ่งมีลอจิกการทำงานดังนี้:
1. **Auto-Login:** เมื่อเปิดแอปพลิเคชันผ่าน LINE ระบบจะดึงรหัสรหัสลับประจำตัว `line_uid` ของแอปพลิเคชัน LINE อัตโนมัติ หากรหัสนั้นเคยผูกไว้ในฐานข้อมูลชีต `staff` แล้ว ระบบจะอนุญาตให้ข้ามหน้าจอลงทะเบียนและเข้าสู่เมนูหลักตามสิทธิ์ (Role-Based Access Control) ทันทีภายใน 1 วินาที
2. **First-Time Login:** หากเป็นผู้ใช้งานใหม่หรือยังไม่เคยผูกบัญชี LINE ระบบจะเปิดหน้าจอให้กรอกรหัส **MDID (รหัส ว. หรือเบอร์มือถือตัวเลข 4-10 หลัก)** * หากรหัส MDID ตรงกับฐานข้อมูลและยังไม่มีใครผูกบัญชี ระบบจะทำการล็อกรหัส `line_uid` และชื่อ `line_displayname` เข้ากับแถวข้อมูลนั้นทันที
   * หากรหัส MDID นั้นเคยถูกผู้อื่นผูกไปแล้ว ระบบจะแจ้งเตือนความปลอดภัยข้อความสีแดงเพื่อป้องกันสิทธิ์ พร้อมเสนอลิงก์ให้กดแจ้งความประสงค์ขอรีเซ็ตสิทธิ์การผูกบัญชีผ่านอีเมลที่ลงทะเบียนไว้

## 2. 📅 การนำเข้าและซิงค์ตารางกิจกรรม (Google Calendar Sync)
ท่านสามารถเชื่อมต่อตารางกิจกรรมของสถาบันเข้าสู่ปฏิทินส่วนตัวบนสมาร์ทโฟนได้อัตโนมัติ ระบบจะอัปเดตวัน-เวลาให้เองหากมีการเปลี่ยนแปลง

1. ไปที่ปุ่ม **"📖 คู่มือการใช้งานระบบ"** ในหน้าเมนูหลัก
2. ในหน้าต่างคู่มือ จะมีกล่องข้อความสำหรับซิงค์ปฏิทิน ให้กดปุ่ม **"➕ กดเพื่อติดตั้ง/ติดตามปฏิทิน"**
3. **วิธีเปิดแสดงผลบนสมาร์ทโฟน:**
   * **สำหรับ Android (แอป Google Calendar):** เปิดแอป > ไปที่เมนู 3 ขีด (มุมซ้ายบน) > การตั้งค่า > กดชื่อปฏิทินสถาบัน > เปิดสวิตช์ "ซิงค์ (Sync)"
   * **สำหรับ iPhone (แอปปฏิทินติดเครื่อง):** หลังจากกดปุ่มติดตั้งแล้ว ให้เข้าเบราว์เซอร์ไปที่ `calendar.google.com/calendar/syncselect` แล้วติ๊กถูกหน้าชื่อปฏิทินสถาบันเพื่อเปิดระบบดาวน์โหลดข้อมูลครับ

## 👨‍⚕️ ส่วนที่ 1: คู่มือการใช้งานสำหรับ Resident / Fellow (แพทย์ประจำบ้านและแพทย์ต่อยอด)

สิทธิ์การใช้งานสำหรับผู้เรียน (Trainee) ออกแบบมาเพื่อเน้นการบันทึกข้อมูลหน้างานและการติดตามพัฒนาการของตนเองอย่างโปร่งใส ประกอบด้วย 5 ฟีเจอร์หลัก:

### 1. การลงชื่อเข้าร่วมกิจกรรมวิชาการ (Academic Activity Check-in)
เมื่อกดปุ่ม **"📍 ลงชื่อเข้าร่วมกิจกรรม"** ระบบจะประมวลผลจัดเรียงรายการกิจกรรมวิชาการที่เปิดอยู่ในสัปดาห์นั้นตามเวลาเริ่มจัดงานจากใกล้ที่สุดไปไกลที่สุด โดยแบ่งรูปแบบการตรวจสอบความปลอดภัยเป็น 2 ประเภทหลัก:
* **กิจกรรมแบบ On-site ล้วน (เรียนที่ห้องเรียน):** หน้าจอจะแสดงปุ่มสีเขียว **"📍 ลงชื่อ On-site"** เมื่อกดปุ่ม ระบบจะส่งคำขอพิกัดดาวเทียม (GPS Geolocation) จากสมาร์ทโฟนของผู้ใช้งาน เพื่อส่งไปตรวจสอบระยะรัศมีหน้าห้องเรียนบนฐานข้อมูลหลังบ้านทันทีโดยไม่ต้องกรอกรหัสใดๆ
* **กิจกรรมแบบ Hybrid / Online (มีลิงก์ Zoom/Webex):** หน้าจอจะแสดงปุ่มสีฟ้า **"ลงชื่อเข้าร่วม (ใช้ PIN)"** พร้อมแสดงปุ่มทางเข้าห้องเรียนออนไลน์สีน้ำเงิน **"📹 เข้าร่วมออนไลน์ (Zoom)"** ผู้เรียนจะต้องกรอกรหัสตัวเลข **PIN 4 หลัก** ที่อาจารย์ผู้สอนประกาศภายในคาบเรียนให้ถูกต้องเพื่อยืนยันการเข้าเรียน 
* *หมายเหตุ:* หากดึงพิกัด GPS ล้มเหลวเนื่องจากสัญญาณอับ ระบบจะขึ้นหน้าต่างคำเตือนแต่จะอนุญาตให้ลงชื่อแบบ "ไม่ระบุพิกัด" เพื่อรักษาประโยชน์ของผู้เรียน

### 2. การสร้างฟอร์มขอรับการประเมินหัตถการ EPA
เมื่อผู้เรียนปฏิบัติหัตถการหรือตรวจเคสที่แผนกผู้ป่วยนอก/ในเสร็จสิ้น สามารถส่งฟอร์มให้อาจารย์ประเมินได้ทันทีโดยกดปุ่ม **"📝 สร้างฟอร์ม EPA"**:
1. **เลือกรายการ EPA:** เลือกหัวข้อหัตถการที่ต้องการรับการประเมิน (ระบบจะแสดงรหัสและรายละเอียดชัดเจน)
2. **เลือกอาจารย์ผู้ประเมิน (Smart Filter):** ระบบจะแสดงรายชื่ออาจารย์แพทย์ที่มีสิทธิ์ประเมิน *พิเศษ:* ระบบมีกลไกดักจับโควต้า หากอาจารย์ท่านใดได้รับการส่งประเมินในหัวข้อนั้นๆ ครบตามเกณฑ์สูงสุดแล้ว (`maxEval`) รายชื่ออาจารย์ท่านนั้นจะถูกเปลี่ยนสถานะเป็น **"โควต้าเต็ม"** เพื่อกระจายการประเมินอย่างเป็นธรรม
3. **บันทึก Note รายละเอียดหัตถการ:** ผู้เรียน **จำเป็นต้องกรอกรายละเอียดเคส** (เช่น ข้อมูลโรคของผู้ป่วย, เทคนิคการผ่าตัด หรือข้อจำกัดหน้างาน) ในช่อง Textarea ห้ามเว้นว่างเด็ดขาด ก่อนกดปุ่ม "บันทึกส่ง" เพื่อส่งคำขอไปยังกล่องงานของอาจารย์ท่านนั้น

### 3. การติดตามประวัติและการดูความก้าวหน้า (My Progress & History)
* **ดูประวัติการส่ง EPA:** แสดงผลรายการคำขอทั้งหมดแยกแถบสีตามสถานะอย่างชัดเจน แถบสีส้มคือ **"⏳ รออาจารย์ประเมิน"**, แถบสีเขียวคือ **"✅ PASS"** (ผ่านเกณฑ์ระดับความรับผิดชอบขั้นต่ำ), และแถบสีแดงคือ **"❌ FAIL"** (ไม่ผ่านเกณฑ์ขั้นต่ำพร้อมระบุเหตุผลและข้อคิดเห็นของอาจารย์เพื่อการปรับปรุง)
* **ดูความก้าวหน้าของฉัน (My Progress Dashboard):** แสดงผลการ์ดเปอร์เซ็นต์ความสำเร็จโดยรวมของการเก็บเคสและการเข้าเรียนเทียบกับเป้าหมายของหลักสูตรประจำปีการศึกษา โดยระบบจะคำนวณ **ความเร็วในการเก็บเคสเทียบกับเวลาที่ผ่านไปจริง (นับตั้งแต่วันที่ 1 กรกฎาคม)** หากอัตราการสะสมเคสช้ากว่าเกณฑ์เวลาอย่างมีนัยสำคัญ ระบบจะเปิดการทำงานของแถบแจ้งเตือนภัยสีแดงระดับวิกฤตความเสี่ยงล่วงหน้า: **"⚠️ แจ้งเตือน: อัตราสะสมเคสช้ากว่าเวลาที่ผ่านไป (X%) กรุณาเร่งเก็บเคสเพิ่ม"**

### 4. การเข้าดูผลสะท้อนกลับรอบทิศทาง (360° Anonymous Feedback)
เมื่อผู้เรียนกดเข้าเมนู **"📊 ดูผลตอบรับ 360"** เพื่ออ่านคะแนนและข้อเสนอแนะที่เพื่อนร่วมงาน อาจารย์ และพยาบาลประเมินให้ ระบบจะปิดบังชื่อผู้ประเมินทั้งหมด (Anonymous) โดยแยกข้อมูลสรุปเป็น 4 กลุ่มผู้ประเมิน (อาจารย์แพทย์, เพื่อนแพทย์ประจำบ้าน, พยาบาลวิชาชีพ, เจ้าหน้าที่/สายสนับสนุน) แสดงคะแนนเฉลี่ย 3 ด้าน (Teamwork, Communication, Punctuality) เต็ม 5 คะแนน พร้อมแยกแท็บข้อมูลระหว่าง **"เทอมปัจจุบัน"** และ **"ประวัติย้อนหลังในแต่ละเทอม"** เพื่อให้เห็นแนวโน้มการพัฒนาตนเอง
* **🚨 เงื่อนไขความปลอดภัยและจริยธรรมสูงสุด (Advisor Gatekeeper):** ผู้เรียนจะสามารถมองเห็นข้อความและคะแนนประเมิน 360 องศาประจำเทอมได้ **ก็ต่อเมื่ออาจารย์ที่ปรึกษา (Advisor) ประจำตัวของตนเอง ได้ทำการกดยืนยันตรวจสอบและอนุมัติ (Allow) ข้อความชิ้นนั้นผ่านระบบหลังบ้านแล้วเท่านั้น** หากอาจารย์ที่ปรึกษายังไม่ได้กดผ่านสิทธิ์ รายการประเมินนั้นจะถูกซ่อนไว้เป็นความลับเพื่อความปลอดภัยทางด้านจิตวิทยาของผู้เรียนเองอีกครั้ง

## 📱 วิธีติดตั้งแอป EPA Skin ลงบนหน้าจอมือถือ (Home Screen)
เพื่อความสะดวกในการเข้าใช้งานครั้งต่อไปแบบเต็มจอ (Full Screen) โดยไม่ต้องเข้าผ่านแอป LINE ท่านสามารถติดตั้งระบบไว้บนหน้าจอมือถือได้ตามขั้นตอนดังนี้:

### 🍎 สำหรับ iOS (iPhone / iPad)
เนื่องจากระบบความปลอดภัยของ LINE ท่านต้องนำลิงก์ไปเปิดใน Safari ด้วยตนเองดังนี้:
1. เมื่อกดลิงก์เข้ามาแล้ว (ในแอป LINE) ให้แตะที่ไอคอน **3 จุด (⋮)** ที่มุมขวาบนสุด
2. เลือกเมนู **"คัดลอกลิงก์" (Copy Link)** 
3. ออกจากแอป LINE แล้วไปเปิดแอป **Safari** ที่หน้าจอ iPhone ของท่าน
4. วางลิงก์ที่คัดลอกมาลงในช่องค้นหา (Address Bar) ด้านบน แล้วกดเปิดเว็บ
5. เมื่อหน้าเว็บโหลดเสร็จ ให้แตะไอคอน **"แชร์" (Share)** ⍗ (รูปสี่เหลี่ยมมีลูกศรชี้ขึ้น) ที่แถบด้านล่าง
6. เลื่อนหาและเลือกเมนู **"เพิ่มไปยังหน้าจอโฮม" (Add to Home Screen)** 
7. กดปุ่ม **"เพิ่ม" (Add)** มุมขวาบน *(ไอคอนแอปจะไปปรากฏอยู่ที่หน้าจอหลักทันที)*

---

### 🤖 สำหรับ Android
1. เมื่อกดลิงก์เข้ามาแล้ว (ในแอป LINE) ให้แตะที่ไอคอน **3 จุด (⋮)** ที่มุมขวาบนสุด
2. เลือกเมนู **"เปิดในเบราว์เซอร์เริ่มต้น"** หรือ **"Open in Browser"** (หน้าเว็บจะเด้งไปเปิดใน Google Chrome)
3. เมื่ออยู่ใน Chrome ให้แตะไอคอนเมนู **3 จุด (⋮)** ที่มุมขวาบนของหน้าจออีกครั้ง
4. เลือกเมนู **"เพิ่มลงในหน้าจอหลัก" (Add to Home screen)**
5. กดปุ่ม **"เพิ่ม" (Add)** เพื่อยืนยันการติดตั้ง

---

### 🤖 สำหรับ Android
1. เปิดลิงก์ระบบนี้ด้วยแอป **Google Chrome** *(หากท่านเปิดผ่านหน้าต่างของ LINE ให้กดปุ่มสามจุดมุมขวาบน -> เลือก "เปิดในเบราว์เซอร์เริ่มต้น" หรือ "Open in Browser")*
2. แตะที่ไอคอนเมนู **3 จุด (⋮)** ที่มุมขวาบนของหน้าจอ Chrome
3. เลือกเมนู **"เพิ่มลงในหน้าจอหลัก" (Add to Home screen)**
4. กดปุ่ม **"เพิ่ม" (Add)** เพื่อยืนยันการติดตั้ง 
*(ระบบจะสร้างไอคอนแอปพลิเคชันไว้บนหน้าจอหลักของโทรศัพท์ให้โดยอัตโนมัติ)*
