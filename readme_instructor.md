# คู่มือการใช้งานระบบ skinEPA เวอร์ชั่น 2.20 (Comprehensive User Manual)

ระบบ **skinEPA Tracker V 2.20** เป็นแพลตฟอร์มดิจิทัลที่พัฒนาขึ้นบนโครงสร้างเทคโนโลยี **LINE Front-end Framework (LIFF)** เชื่อมต่อฐานข้อมูลประสิทธิภาพสูงแบบ Real-time ร่วมกับ **Google Apps Script (GAS)** และ **Google Sheets** มีวัตถุประสงค์หลักเพื่อบันทึก ประเมิน และติดตามความก้าวหน้าทางการศึกษาของแพทย์ประจำบ้าน (Resident) และแพทย์ประจำบ้านต่อยอด (Fellow) ทั้งในส่วนของหัตถการหลัก (EPA), การเข้าร่วมกิจกรรมวิชาการ (Academic Attendance) และการประเมินผลรอบทิศทาง 360 องศา โดยเวอร์ชั่น 2.20 นี้ได้พัฒนาโครงสร้างความปลอดภัย ระบบคัดกรองข้อมูลโดยอาจารย์ที่ปรึกษา และระบบสื่อสารความเร็วสูงผ่าน LINE API อย่างสมบูรณ์

---

## 🔐 โครงสร้างการเข้าสู่ระบบและการยืนยันตัวตน (Authentication Flow)
ระบบใช้กลไกการยืนยันตัวตนแบบ **LINE UID Binding Cross-Reference** ควบคุมป๊อปอัปแจ้งเตือนทั้งหมดด้วยโปรแกรม **SweetAlert2 (Swal)** ซึ่งมีลอจิกการทำงานดังนี้:
1. **Auto-Login:** เมื่อเปิดแอปพลิเคชันผ่าน LINE ระบบจะดึงรหัสรหัสลับประจำตัว `line_uid` ของแอปพลิเคชัน LINE อัตโนมัติ หากรหัสนั้นเคยผูกไว้ในฐานข้อมูลชีต `staff` แล้ว ระบบจะอนุญาตให้ข้ามหน้าจอลงทะเบียนและเข้าสู่เมนูหลักตามสิทธิ์ (Role-Based Access Control) ทันทีภายใน 1 วินาที
2. **First-Time Login:** หากเป็นผู้ใช้งานใหม่หรือยังไม่เคยผูกบัญชี LINE ระบบจะเปิดหน้าจอให้กรอกรหัส **MDID (รหัส ว. หรือเบอร์มือถือตัวเลข 4-10 หลัก)** * หากรหัส MDID ตรงกับฐานข้อมูลและยังไม่มีใครผูกบัญชี ระบบจะทำการล็อกรหัส `line_uid` และชื่อ `line_displayname` เข้ากับแถวข้อมูลนั้นทันที
   * หากรหัส MDID นั้นเคยถูกผู้อื่นผูกไปแล้ว ระบบจะแจ้งเตือนความปลอดภัยข้อความสีแดงเพื่อป้องกันสิทธิ์ พร้อมเสนอลิงก์ให้กดแจ้งความประสงค์ขอรีเซ็ตสิทธิ์การผูกบัญชีผ่านอีเมลที่ลงทะเบียนไว้

## 2. 📅 การนำเข้าและซิงค์ตารางกิจกรรม (Google Calendar Sync)
อาจารย์สามารถเชื่อมต่อตารางกิจกรรมของสถาบันเข้าสู่ปฏิทินส่วนตัวบนสมาร์ทโฟนได้อัตโนมัติ ระบบจะอัปเดตวัน-เวลาให้เองหากมีการเปลี่ยนแปลง

1. ไปที่ปุ่ม **"📖 คู่มือการใช้งานระบบ"** ในหน้าเมนูหลัก
2. ในหน้าต่างคู่มือ จะมีกล่องข้อความสำหรับซิงค์ปฏิทิน ให้กดปุ่ม **"➕ กดเพื่อติดตั้ง/ติดตามปฏิทิน"**
3. **วิธีเปิดแสดงผลบนสมาร์ทโฟน:**
   * **สำหรับ Android (แอป Google Calendar):** เปิดแอป > ไปที่เมนู 3 ขีด (มุมซ้ายบน) > การตั้งค่า > กดชื่อปฏิทินสถาบัน > เปิดสวิตช์ "ซิงค์ (Sync)"
   * **สำหรับ iPhone (แอปปฏิทินติดเครื่อง):** หลังจากกดปุ่มติดตั้งแล้ว ให้เข้าเบราว์เซอร์ไปที่ `calendar.google.com/calendar/syncselect` แล้วติ๊กถูกหน้าชื่อปฏิทินสถาบันเพื่อเปิดระบบดาวน์โหลดข้อมูลครับ

## 👨‍🏫 ส่วนที่ 2: คู่มือการใช้งานสำหรับ Instructor (อาจารย์แพทย์ / อาจารย์ที่ปรึกษา)

สิทธิ์การใช้งานของอาจารย์แพทย์เน้นความสะดวก รวดเร็วในการประเมินคะแนนข้างเตียงผู้ป่วย และทำหน้าที่เป็นที่ปรึกษาคัดกรองข้อมูลพัฒนาการของลูกศิษย์

### 1. การประเมินฟอร์มหัตถการแพทย์ประจำบ้าน (EPA Evaluation Task)
เมื่อลูกศิษย์ส่งฟอร์มคำขอประเมินเข้ามา อาจารย์จะได้รับการแจ้งเตือนและสามารถกดเข้าเมนู **"✅ รอประเมิน EPA"** เพื่อดำเนินการ:
1. หน้าจอจะแสดงรายการคำขอค้างประเมินทั้งหมด ระบุชื่อแพทย์ประจำบ้าน ชั้นปี วันเวลาที่ทำหัตถการ และข้อความ Note รายละเอียดเคสที่เด็กพิมพ์แนบมา
2. อาจารย์คลิกเลือกเมนูคะแนน **EPA Level (ระดับ 1 ถึง 5)** ตามเกณฑ์สากล:
   * *Level 1:* ดูอย่างเดียว (Observe)
   * *Level 2:* อาจารย์คุมและช่วยทำ (Direct Supervision)
   * *Level 3:* ทำเองโดยมีอาจารย์คุมอยู่ใกล้ๆ (Indirect Supervision)
   * *Level 4:* ทำเองได้คนเดียวอย่างปลอดภัย (Unsupervised)
   * *Level 5:* ชำนาญจนสามารถสอนและคุมน้องทำได้ (Aspirant/Teacher)
3. **ระบบตรวจสอบเกณฑ์ผ่านขั้นต่ำอัตโนมัติ (Automated Competency Check):** หากอาจารย์เลือกให้คะแนน **ต่ำกว่าเกณฑ์ขั้นต่ำ (`minLevel`)** ที่หลักสูตรกำหนดไว้ในสัญญารายปีของชั้นปีนั้นๆ ระบบจะเปิดแผงควบคุมข้อผิดพลาดสีแดงบังคับให้อาจารย์ **เลือกติ๊กเหตุผลที่ไม่ผ่านเกณฑ์จาก Checkbox ขั้นต่ำ 1 ข้อ** (เช่น เทคนิคยังไม่แม่นยำ, การแก้ไขปัญหาเฉพาะหน้า หรือการดูแลผู้ป่วยหลังผ่าตัด) พร้อมทั้งเปิดช่องพิมพ์ความเห็นเพิ่มเติมเพื่อส่งสถิติกลับไปแจ้งเตือนผู้เรียน
4. กดปุ่ม **"ยืนยันผล"** เพื่อบันทึกข้อมูลเข้าสู่ระบบทันที การ์ดงานจะสไลด์หายไปอัตโนมัติ

### 2. 👨‍🏫 ระบบกล่องข้อความอาจารย์ที่ปรึกษา (Advisor Approval UI) - ฟีเจอร์ใหม่
เป็นหน้าต่างที่พัฒนาขึ้นมาเฉพาะเพื่อให้อาจารย์ที่คุมทีมและเป็นที่ปรึกษา สามารถดูแลสภาพจิตใจและพฤติกรรมของเด็กในความดูแลได้อย่างใกล้ชิด โดยเมื่ออาจารย์กดปุ่ม **"👨‍🏫 กล่องข้อความอาจารย์ที่ปรึกษา (คัดกรอง 360°)"** ระบบจะทำงานดังนี้:
1. ระบบดึงรายชื่อแพทย์ประจำบ้านทุกคนในสถาบันที่มีรหัสของอาจารย์ระบุอยู่ในคอลัมน์ `advisor_mdid` (ชีต `staff` รองรับการใส่อาจารย์ที่ปรึกษาร่วมหลายคนคั่นด้วยเครื่องหมายจุลภาค)
2. ระบบจะดึงฟอร์มประเมิน 360 องศาของเด็กกลุ่มดังกล่าวที่ส่งเข้ามาในเทอมนั้นและ **ยังค้างสถานะรอตรวจสอบ** ขึ้นมาแสดงผลทีละฉบับ โดยอาจารย์จะเห็นบทบาทของผู้ประเมิน (เช่น พยาบาล หรือเพื่อนร่วมงาน), คะแนนดิบ 3 ด้าน และข้อความวิจารณ์ (Feedback Text) ทั้งหมดอย่างละเอียด
3. **การตัดสินใจ (Action):** อาจารย์ต้องพิจารณาเนื้อหาและข้อคิดเห็นนั้นๆ:
   * **กดปุ่ม "✅ อนุญาตให้เห็น (Allow)":** หากพิจารณาแล้วว่าข้อความนั้นเป็นประโยชน์ เป็น Constructive Feedback ที่เด็กควรรับทราบเพื่อปรับปรุงตัว ระบบจะเปลี่ยนสถานะตัวบ่งชี้ในชีตเป็น `approved` และปล่อยข้อความชิ้นนั้นให้ไปปรากฏบนหน้าจอมือถือของแพทย์ประจำบ้านคนนั้นทันที
   * **กดปุ่ม "❌ ซ่อนไว้ (Reject)":** หากพิจารณาแล้วว่าข้อความใช้ถ้อยคำที่รุนแรงเกินไป บั่นทอนจิตใจ หรือไม่เกิดประโยชน์ทางการศึกษา ระบบจะเปลี่ยนสถานะเป็น `rejected` ข้อความจะถูกล็อกเก็บไว้ในฐานข้อมูลแอดมินเพื่อการตรวจสอบ แต่จะไม่แสดงผลให้นักเรียนเห็นเด็ดขาด

### 3. การติดตามพัฒนาการเชิงลึกของลูกศิษย์ (Advisee Portfolio Viewer)
อาจารย์แพทย์สามารถเปิดดูสมุดพกความก้าวหน้าด้านการเรียนรู้รอบทิศทางของศิษย์ในความดูแลได้โดยกดปุ่ม **"📊 ดูผลประเมิน 360 (ลูกศิษย์)"**:
* เลือกรายชื่อเด็กในความดูแลจาก Dropdown Menu
* ระบบจะคำนวณประมวลผล **สัดส่วนและที่มาของฟอร์มประเมินในแต่ละเทอม (Summary Breakdown)** สรุปให้เห็นว่าในเทอมนั้นๆ มีแบบประเมินมาจากอาจารย์ เพื่อน พยาบาล และเจ้าหน้าที่กลุ่มละกี่ฉบับอย่างชัดเจน
* แสดง **ตารางเปรียบเทียบพัฒนาการย้อนหลัง (Historical Longitudinal Table)** ดึงคะแนนเฉลี่ยภาพรวมแต่ละด้านในแต่ละเทอมมาจัดเรียงเปรียบเทียบกันเป็น Timeline พร้อมไฮไลท์สีตามระดับคะแนน (คะแนนสูงกว่า 4.0 เป็นสีเขียว, คะแนนมาตรฐานเป็นสีฟ้า, คะแนนต่ำกว่า 3.0 เป็นสีแดง) เพื่อให้อาจารย์เห็นจุดบกพร่องและแนวโน้มพฤติกรรมของนักเรียนได้อย่างแม่นยำ

## 📱 วิธีติดตั้งแอป EPA Skin ลงบนหน้าจอมือถือ (Home Screen)
เพื่อความสะดวกในการเข้าใช้งานครั้งต่อไปแบบเต็มจอ (Full Screen) โดยไม่ต้องเข้าผ่านแอป LINE ท่านสามารถติดตั้งระบบไว้บนหน้าจอมือถือได้ตามขั้นตอนดังนี้:

### 🍎 สำหรับ iOS (iPhone / iPad)
เนื่องจากระบบความปลอดภัยของ LINE ท่านต้องนำลิงก์ไปเปิดใน Safari ด้วยตนเองดังนี้:
1. เมื่อกดลิงก์เข้ามาแล้ว (ในแอป LINE) ให้แตะที่ไอคอน **3 จุด (⋮)** ที่มุมขวาบนสุด
2. เลือกเมนู **"คัดลอกลิงก์" (Copy Link)** 
3. ออกจากแอป LINE แล้วไปเปิดแอป **Safari** ที่หน้าจอ iPhone ของท่าน
4. วางลิงก์ที่คัดลอกมาลงในช่องค้นหา (Address Bar) ด้านบน แล้วกดเปิดเว็บ
5. เมื่อหน้าเว็บโหลดเสร็จ ให้แตะไอคอน **"แชร์" (Share)** ⍗ (รูปสี่เหลี่ยมมีลูกศรชี้ขึ้น) ที่แถบด้านล่าง
6. เลื่อนหาและเลือกเมนู **"เพิ่มไปยังหน้าจอโฮม" (Add to Home Screen)** 
7. กดปุ่ม **"เพิ่ม" (Add)** มุมขวาบน *(ไอคอนแอปจะไปปรากฏอยู่ที่หน้าจอหลักทันที)*

---

### 🤖 สำหรับ Android
1. เมื่อกดลิงก์เข้ามาแล้ว (ในแอป LINE) ให้แตะที่ไอคอน **3 จุด (⋮)** ที่มุมขวาบนสุด
2. เลือกเมนู **"เปิดในเบราว์เซอร์เริ่มต้น"** หรือ **"Open in Browser"** (หน้าเว็บจะเด้งไปเปิดใน Google Chrome)
3. เมื่ออยู่ใน Chrome ให้แตะไอคอนเมนู **3 จุด (⋮)** ที่มุมขวาบนของหน้าจออีกครั้ง
4. เลือกเมนู **"เพิ่มลงในหน้าจอหลัก" (Add to Home screen)**
5. กดปุ่ม **"เพิ่ม" (Add)** เพื่อยืนยันการติดตั้ง
*หากพบปัญหาการใช้งาน กรุณาติดต่อเจ้าหน้าที่ดูแลระบบของหน่วย/ภาควิชาฯ*
