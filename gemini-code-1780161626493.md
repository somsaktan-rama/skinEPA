# ⚙️ คู่มือการใช้งานระบบ EPA Skin Tracker (อัปเดตระบบ Hybrid Learning)
**สำหรับ พยาบาลวิชาชีพ, นักวิทยาศาสตร์, Clerk และ เจ้าหน้าที่สายสนับสนุน (Staff / Admin)**

ระบบ EPA Skin Tracker ถูกออกแบบมาเพื่อช่วยให้เจ้าหน้าที่และ Admin สามารถจัดการกิจกรรมวิชาการ และติดตามความก้าวหน้าของการประเมินของแพทย์ประจำบ้านได้อย่างมีประสิทธิภาพ

---

## 1. 🔑 การเข้าสู่ระบบครั้งแรก (Login)
1. เปิดแอปพลิเคชันผ่านลิงก์ที่ได้รับ
2. กรอก **รหัส MDID** (รหัส ว. หรือ เบอร์โทรศัพท์ 4-10 หลัก) ในช่องเข้าสู่ระบบ
3. กด **เข้าสู่ระบบ** ระบบจะผูกบัญชี LINE (LINE Binding) ของท่านเข้ากับระบบอัตโนมัติ
4. การเข้าใช้งานครั้งต่อไป ระบบจะจดจำและเข้าสู่ระบบให้ท่านทันที (Auto-login)

---

## 2. 📅 การนำเข้าและซิงค์ตารางกิจกรรม (Google Calendar Sync)
ท่านสามารถเชื่อมต่อตารางกิจกรรมของสถาบันเข้าสู่ปฏิทินส่วนตัวบนสมาร์ทโฟนได้อัตโนมัติ ระบบจะอัปเดตวัน-เวลาให้เองหากมีการเปลี่ยนแปลง

1. ไปที่ปุ่ม **"📖 คู่มือการใช้งานระบบ"** ในหน้าเมนูหลัก
2. ในหน้าต่างคู่มือ จะมีกล่องข้อความสำหรับซิงค์ปฏิทิน ให้กดปุ่ม **"➕ กดเพื่อติดตั้ง/ติดตามปฏิทิน"**
3. **วิธีเปิดแสดงผลบนสมาร์ทโฟน:**
   * **สำหรับ Android (แอป Google Calendar):** เปิดแอป > ไปที่เมนู 3 ขีด (มุมซ้ายบน) > การตั้งค่า > กดชื่อปฏิทินสถาบัน > เปิดสวิตช์ "ซิงค์ (Sync)"
   * **สำหรับ iPhone (แอปปฏิทินติดเครื่อง):** หลังจากกดปุ่มติดตั้งแล้ว ให้เข้าเบราว์เซอร์ไปที่ `calendar.google.com/calendar/syncselect` แล้วติ๊กถูกหน้าชื่อปฏิทินสถาบันเพื่อเปิดระบบดาวน์โหลดข้อมูลครับ

## 3. 📝 การจัดการกิจกรรมวิชาการ (Activity Management)
เจ้าหน้าที่ธุรการหรือ Admin สามารถอำนวยความสะดวกให้ผู้เรียนและอาจารย์ โดยการสร้างและจัดการกิจกรรม รวมถึงแนบลิงก์การประชุมออนไลน์ (Zoom/Webex) ได้ดังนี้:

### 3.1 การเปิดกิจกรรมใหม่ (แบบครั้งเดียว)
1. ไปที่เมนู **เปิดกิจกรรมใหม่ (แบบครั้งเดียว)**
2. เลือกรหัสกิจกรรม, พิมพ์หัวข้อ (Topic), และระบุอาจารย์ผู้คุม (ถ้ามี)
3. **ลิงก์ประชุม (Zoom/Webex):** ท่านสามารถนำ URL ของห้องประชุมมาวางได้เลย (หากเป็นคลาส On-site ปกติ ให้เว้นว่างไว้)
4. เลือกวัน-เวลาเริ่มต้น และระยะเวลา (เช่น 60 นาที) แล้วกด **ยืนยันสร้าง**

### 3.2 ระบบแม่แบบกิจกรรมรายสัปดาห์ (Weekly Batch)
ฟีเจอร์นี้เหมาะสำหรับกิจกรรมที่มีตารางเวลาซ้ำๆ กันทุกสัปดาห์ เพื่อลดภาระการสร้างใหม่
* **สร้างแม่แบบใหม่:** ไปที่เมนู **ระบบแม่แบบรายสัปดาห์** กรอกข้อมูลกิจกรรม เลือกวันในสัปดาห์ และท่านสามารถ **แนบลิงก์การประชุมออนไลน์** เข้าไปในช่องแม่แบบ เพื่อให้ระบบจดจำลิงก์ประจำวิชานั้นๆ ไว้ใช้ในสัปดาห์ถัดๆ ไปได้ แล้วกด **"บันทึกแม่แบบ"**
* **การสร้างลงตารางจริง:** ติ๊กเครื่องหมายถูก (✔️) หน้าแม่แบบที่ต้องการ แล้วกดปุ่ม **"ยืนยันสร้างกิจกรรมที่เลือก"** ระบบจะนำกิจกรรมนั้นไปสร้างลงในสัปดาห์ถัดไปให้อัตโนมัติ
* **การลบแม่แบบ:** สามารถกดปุ่มถังขยะ (🗑️) เพื่อลบแม่แบบที่ไม่ต้องการใช้งานแล้วได้

### 3.3 การแก้ไขและจัดการกิจกรรมที่เปิดอยู่ (Manage Activities)
1. ไปที่เมนู **จัดการกิจกรรม**
2. คลิกที่กิจกรรมที่กำลังเปิดอยู่ (สถานะ OPEN)
3. **สิ่งที่ท่านสามารถจัดการได้:**
   * **✏️ แก้ไข (Edit):** ปรับเปลี่ยนหัวข้อ เปลี่ยนตัวอาจารย์ผู้ดูแล อัปเดต/เปลี่ยนลิงก์ Zoom ใหม่ หรือแก้ไขวัน-เวลาเริ่มและจบกิจกรรมได้ (ระบบจะอนุญาตให้แก้ได้หากกิจกรรมยังมีสถานะเป็น OPEN)
   * **นำออก (Remove Attendee):** หากพบผู้ที่ลงชื่อผิดหรือไม่ได้เข้าร่วมกิจกรรมจริง สามารถกดนำชื่อออกได้
   * **ปิดกิจกรรม (Close):** เมื่อสิ้นสุดกิจกรรมแล้ว สามารถกดเพื่อปิดรับการลงชื่อได้ (ระบบป้องกันไม่ให้ปิดก่อนเวลาสิ้นสุดที่ตั้งไว้)
   * **🗑️ ลบกิจกรรม (Delete):** กดเพื่อลบกิจกรรมและข้อมูลการลงชื่อทั้งหมดออกจากระบบ (ใช้ในกรณียกเลิกการจัดกิจกรรม)

## 4. 📍 ระบบลงชื่อเข้าร่วมกิจกรรม (Activity Check-in ด้วย GPS)
ระบบรองรับการลงชื่อเข้าร่วมกิจกรรมผ่านสมาร์ทโฟน พร้อมการตรวจสอบพิกัด (GPS) แบบยืดหยุ่น โดยมีเงื่อนไขการทำงานดังนี้:

### 4.1 เงื่อนไขเวลาการลงชื่อ
* ผู้เข้าร่วมสามารถกด Check-in ได้ตั้งแต่ **30 นาทีก่อนเวลาเริ่มกิจกรรม (act_begin)** ไปจนถึง **เวลาสิ้นสุดกิจกรรม (act_finish)**
* หากอยู่นอกเหนือช่วงเวลานี้ ปุ่มลงชื่อจะถูกล็อก (ตัวเทา) อัตโนมัติ

### 4.2 การตรวจสอบพิกัดแบบ Dynamic (อ้างอิงจากอาจารย์ผู้คุม)
ระบบไม่ได้ล็อกพิกัดสถานที่แบบตายตัว แต่จะใช้ **"พิกัดของอาจารย์ผู้คุมกิจกรรม"** เป็นจุดศูนย์กลาง เพื่อรองรับการจัดกิจกรรมนอกสถานที่หรือในวอร์ดต่างๆ โดยมีสถานะป้ายสีในหน้ารายชื่อผู้เข้าร่วมดังนี้:
* 🟪 **👨‍🏫 อาจารย์ผู้คุม:** แสดงสถานะของอาจารย์ที่เป็นผู้รับผิดชอบกิจกรรมนั้น (ระบบจะใช้พิกัดนี้เป็นศูนย์กลางรัศมี 1 กิโลเมตร)
* 🟦 **⏳ รอเทียบพิกัดอาจารย์:** เกิดขึ้นเมื่อแพทย์ประจำบ้านกด Check-in ก่อนที่อาจารย์ผู้คุมจะเข้าสู่ระบบ (ระบบบันทึกเวลาและพิกัดไว้แล้ว และจะเปลี่ยนสีให้ทันทีเมื่ออาจารย์ Check-in ตามเข้ามา)
* 🟩 **✔️ ในพื้นที่ (In Range):** ผู้เข้าร่วมกด Check-in และอยู่ห่างจากอาจารย์ผู้คุม **ไม่เกิน 1 กิโลเมตร**
* 🟥 **❌ นอกรัศมี (Out of Range):** ผู้เข้าร่วมกด Check-in แต่อยู่ห่างจากอาจารย์ผู้คุม **มากกว่า 1 กิโลเมตร**
* ⬜ **⚠️ ไม่ระบุพิกัด (No GPS):** กรณีที่สมาร์ทโฟนของผู้ใช้งานไม่อนุญาตให้เข้าถึง Location (หรือหาพิกัดไม่สำเร็จในเวลาที่กำหนด) ระบบจะอนุญาตให้ลงชื่อเพื่อรักษาสิทธิ์ แต่จะขึ้นป้ายแจ้งเตือนสีเทาไว้ เพื่อให้อาจารย์ตรวจสอบด้วยสายตาตนเองอีกครั้ง

> **💡 ข้อแนะนำสำหรับเจ้าหน้าที่:** หากพบผู้เข้าร่วมที่ขึ้นป้าย `❌ นอกรัศมี` หรือ `⚠️ ไม่ระบุพิกัด` ท่านสามารถช่วยอาจารย์พิจารณาเช็คชื่อด้วยสายตาในห้องประชุม และสามารถกดปุ่ม **"นำออก"** เพื่อลบรายชื่อผู้ที่ไม่ได้มาเข้าร่วมจริงออกจากระบบได้ทันที

## 5. 📊 ระบบติดตามสำหรับ Admin (Dashboards)
Admin สามารถดูรายงานสรุปความก้าวหน้าเพื่อติดตามผลการประเมินได้ที่หน้าจอหลัก:

* **📈 Dashboard ความก้าวหน้า EPA:**
  * ใช้ดูภาพรวมการเก็บเคส EPA และการเข้าร่วมกิจกรรมของแพทย์ทุกคนแบบเรียลไทม์
  * ระบบจะแสดงแถบแจ้งเตือนความเสี่ยง (Risk Alert) สีแดง หากแพทย์ท่านใดมีอัตราสะสมเคสช้ากว่าระยะเวลาที่ผ่านไป
  * สามารถคลิกดูรายละเอียดรายบุคคล และกดปุ่ม **"📥 Export Excel"** เพื่อดาวน์โหลดข้อมูลเป็นไฟล์ .csv นำไปทำรายงานต่อได้
* **🕵️‍♂️ ติดตามสถานะประเมิน 360° (Admin Tracking):**
  * ใช้ตรวจสอบยอดรวมจำนวนใบประเมิน 360 องศาที่แพทย์แต่ละคนได้รับ
  * หากรายชื่อใดแสดงผลเป็น **แถบสีแดง** (ได้ยอดรวมน้อยกว่า 5 ฉบับ) Admin ควรดำเนินการติดตามหรือแจ้งเตือนให้อาจารย์และเจ้าหน้าที่ท่านอื่นช่วยดำเนินการประเมินเพิ่มเติม
