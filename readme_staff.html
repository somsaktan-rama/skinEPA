1. การจัดการตารางกิจกรรม (Schedule)

  • Single Session: เปิดกิจกรรมใหม่แบบรายครั้ง ข้อมูลจะถูกบันทึกลงฐานข้อมูลประจำปีการศึกษาที่เลือกไว้โดยอัตโนมัติ
  • Weekly Batch Template: เข้าเมนู "ระบบแม่แบบรายสัปดาห์" เพื่อดึงเทมเพลตมาเปิดคลาสเรียนล่วงหน้าทั้งสัปดาห์ (สูงสุด 10 คลาส) ในคลิกเดียว

2. Live Attendee Tracker (ควบคุมห้องเรียน)

ตรวจสอบพิกัดผู้เข้าเรียนแบบ Real-time ตามสีสถานะภายในปีการศึกษาปัจจุบัน:

พิกัดอาจารย์
อยู่ในระยะ
อยู่นอกพื้นที่
รอเปิดคลาส

หมายเหตุ: ปุ่ม "ลบกิจกรรม" จะทำงานได้ก็ต่อเมื่อยังไม่มีผู้ลงชื่อเข้าเรียนในเซสชันนั้นๆ เท่านั้น

3. แดชบอร์ดติดตามผลเชิงบริหาร (Analytics)

ระบบแดชบอร์ดได้รับการปรับปรุงโครงสร้างให้รองรับการกรองข้อมูลแบบ Multi-Year Selection ผ่านตัวเลือกด้านบนโปรแกรม:

📊 ความก้าวหน้า EPA & กิจกรรมรายปี

  • เมื่อทำการเปลี่ยนตัวเลือกปีการศึกษา แดชบอร์ดจะสลับไปดึงข้อมูลจากฐานข้อมูลของปีนั้นๆ ทันที (Default ระบบจะคำนวณเป็นปีปัจจุบันเสมอ) ช่วยลดภาระการโหลดข้อมูลและการหน่วงของระบบ
  • Export ข้อมูลจำแนกปี: รองรับการกด Export Pivot Excel และ Export Raw Data CSV โดยไฟล์ที่ดาวน์โหลดออกมาจะดึงเฉพาะข้อมูลดิบของปีการศึกษาที่เลือก สะดวกต่อการจัดเก็บสถิติแยกรายปี

🔄 ระบบติดตามยอดประเมิน 360 (Admin View)

เป้าหมายขั้นต่ำประจำปี:
