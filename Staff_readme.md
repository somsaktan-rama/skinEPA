# 👨‍🏫 คู่มือการใช้งานระบบ EPA Skin Tracker
**สำหรับอาจารย์แพทย์ (Instructor)**

ระบบ EPA Skin Tracker v2.0 ถูกออกแบบมาเพื่ออำนวยความสะดวกให้อาจารย์แพทย์ในการประเมินหัตถการ (EPA) ของแพทย์ประจำบ้าน, ติดตามผลประเมิน 360 องศาของแพทย์ในการดูแล และจัดการกิจกรรมวิชาการ ได้อย่างรวดเร็วผ่านแอปพลิเคชัน LINE

---

## 1. 🔑 การเข้าสู่ระบบครั้งแรก (Login & LINE Binding)
เพื่อความสะดวกรวดเร็วในการใช้งาน ระบบจะเชื่อมต่อกับบัญชี LINE ของอาจารย์
1. เปิดแอปพลิเคชันผ่านลิงก์ที่ได้รับในห้องแชท LINE
2. กรอก **รหัส MDID** (รหัส ว. หรือ เบอร์มือถือ 4-10 หลัก) ในช่องเข้าสู่ระบบ
3. กด **เข้าสู่ระบบ** ระบบจะทำการผูกบัญชี LINE (LINE Binding) อัตโนมัติ
4. ในครั้งต่อไปที่เปิดแอปพลิเคชันผ่าน LINE เครื่องเดิม ระบบจะจดจำและเข้าสู่ระบบอัตโนมัติ (Auto-login) ทันที

> **💡 หมายเหตุ:** หากอาจารย์มีการเปลี่ยนโทรศัพท์มือถือ สามารถกดข้อความ **"ต้องการรีเซ็ตการผูกบัญชี LINE?"** ที่หน้าแรก เพื่อให้ระบบส่งลิงก์ปลดล็อกไปยังอีเมลที่ลงทะเบียนไว้ได้ครับ

---

## 2. ✅ การประเมิน EPA (EPA Evaluation)
เมื่อแพทย์ประจำบ้านส่งรายการ EPA มาถึงอาจารย์ ระบบจะส่งข้อความแจ้งเตือนเข้าทาง LINE ส่วนตัวของอาจารย์ทันที
1. ไปที่เมนู **รอประเมิน EPA** ในหน้าจอหลัก
2. ระบบจะแสดงรายการหัตถการที่รอการประเมิน พร้อมกับ **Note** ที่แพทย์ประจำบ้านบันทึกส่งมา
3. คลิกที่รายการที่ต้องการประเมิน ระบบจะแสดงระดับคะแนนที่ต้องผ่าน (Level เกณฑ์ขั้นต่ำ)
4. **เลือกให้คะแนน (Level 1-5):**
   * 1 = ดูอย่างเดียว
   * 2 = อาจารย์คุมให้ทำ
   * 3 = ทำเอง/อาจารย์คุม
   * 4 = ทำเองคนเดียว
   * 5 = สอนน้องทำได้
5. **กรณีให้คะแนน "ไม่ผ่านเกณฑ์":** ระบบจะแสดงหัวข้อให้เลือกเหตุผลที่ไม่ผ่าน และมีช่องให้พิมพ์ข้อเสนอแนะเพิ่มเติมเพื่อเป็น Feedback ให้แพทย์ประจำบ้าน[cite: 4]
6. กด **ยืนยันผล** ข้อมูลจะถูกบันทึกและแจ้งเตือนกลับไปยังแพทย์ประจำบ้านทันที[cite: 4]

---

## 3. 📊 การดูผลประเมิน 360 องศา (ลูกศิษย์ในความดูแล)
อาจารย์สามารถติดตามผลประเมิน 360 องศาของแพทย์ประจำบ้านหรือแพทย์ต่อยอดที่อยู่ในความดูแลของอาจารย์ได้ (อ้างอิงจากรายชื่อที่กำหนดไว้ในระบบ)
1. ไปที่เมนู **ดูผลประเมิน 360 (ลูกศิษย์)**
2. เลือกรายชื่อแพทย์ที่ต้องการดูผลประเมินจาก Dropdown
3. **สิ่งที่ระบบจะแสดงผล:**
   * **สัดส่วนและที่มาของผลประเมิน:** แสดงจำนวนฉบับที่ส่งประเมิน แยกตามกลุ่มผู้ประเมิน (อาจารย์, เพื่อนแพทย์, พยาบาล, สายสนับสนุน)
   * **ตารางเปรียบเทียบพัฒนาการ:** แสดงคะแนนเฉลี่ยใน 3 หัวข้อ (การทำงานเป็นทีม, การสื่อสาร, ความรับผิดชอบ) เปรียบเทียบย้อนหลังในแต่ละเทอม
   * **รวบรวมข้อคิดเห็น:** แสดงข้อเสนอแนะแบบไม่ระบุตัวตน (Anonymous) แยกตามเทอมและกลุ่มผู้ประเมิน เพื่อให้อาจารย์นำไปใช้เป็นข้อมูลประกอบการให้คำปรึกษา

## 4. ⚙️ การจัดการกิจกรรม (Activity Management)
หากอาจารย์ได้รับมอบหมายให้เป็นผู้ดูแลกิจกรรมวิชาการ อาจารย์สามารถจัดการกิจกรรมที่ "กำลังเปิดลงชื่ออยู่" ได้ดังนี้
1. ไปที่เมนู **จัดการกิจกรรม**
2. คลิกที่กิจกรรมที่ต้องการจัดการ ระบบจะแสดงรายชื่อแพทย์และเจ้าหน้าที่ที่กด Check-in เข้ามาแล้ว
3. **เครื่องมือสำหรับจัดการกิจกรรม:**
   * **✏️ แก้ไข (Edit):** สามารถแก้ไขหัวข้อกิจกรรม, เปลี่ยนชื่ออาจารย์ผู้ดูแล, หรือปรับเวลาเริ่มต้น/สิ้นสุดได้ (ทำได้ตราบใดที่กิจกรรมยังไม่ถูกปิด)
   * **นำออก (Remove):** ใช้เตะรายชื่อผู้ที่ไม่ได้เข้าร่วมจริงออกจากระบบ
   * **ปิดกิจกรรม (Close):** เมื่อสิ้นสุดกิจกรรมแล้ว ควรกดปิดกิจกรรม เพื่อป้องกันการลงชื่อย้อนหลัง
   * **🗑️ ลบ (Delete):** ลบกิจกรรมและข้อมูลการลงชื่อทั้งหมดออกจากระบบ (ใช้ในกรณียกเลิกกิจกรรม)

> **⏰ ระบบแจ้งเตือนลืมปิดกิจกรรม:** หากกิจกรรมเลยเวลาสิ้นสุดมาแล้ว 1 ชั่วโมง ระบบจะส่งข้อความแจ้งเตือนเตือนให้อาจารย์หรือ Staff เข้ามากด "ปิดกิจกรรม" ครับ
