# ⚙️ คู่มือการใช้งานระบบ EPA Skin Tracker
**สำหรับ พยาบาลวิชาชีพ, นักวิทยาศาสตร์, Clerk และ เจ้าหน้าที่สายสนับสนุน (Staff / Admin)**

ระบบ EPA Skin Tracker ถูกออกแบบมาเพื่อช่วยให้เจ้าหน้าที่และ Admin สามารถจัดการกิจกรรมวิชาการ และติดตามความก้าวหน้าของการประเมินของแพทย์ประจำบ้านได้อย่างมีประสิทธิภาพ

---

## 1. 🔑 การเข้าสู่ระบบครั้งแรก (Login)
1. เปิดแอปพลิเคชันผ่านลิงก์ที่ได้รับ
2. กรอก **รหัส MDID** (รหัส ว. หรือ เบอร์โทรศัพท์ 4-10 หลัก) ในช่องเข้าสู่ระบบ
3. กด **เข้าสู่ระบบ** ระบบจะผูกบัญชี LINE (LINE Binding) ของท่านเข้ากับระบบอัตโนมัติ
4. การเข้าใช้งานครั้งต่อไป ระบบจะจดจำและเข้าสู่ระบบให้ท่านทันที (Auto-login)

---

## 2. 📅 การจัดการกิจกรรมวิชาการ (Activity Management)
เจ้าหน้าที่ธุรการหรือ Admin สามารถจัดการการลงชื่อเข้าร่วมกิจกรรมวิชาการได้ดังนี้:

### 2.1 การเปิดกิจกรรมใหม่ (แบบครั้งเดียว)
1. ไปที่เมนู **เปิดกิจกรรมใหม่ (แบบครั้งเดียว)**
2. เลือกรหัสกิจกรรม, พิมพ์หัวข้อ (Topic), และระบุอาจารย์ผู้คุม (ถ้ามี)
3. เลือกวัน-เวลาเริ่มต้น และระยะเวลา (เช่น 60 นาที) แล้วกด **ยืนยันสร้าง**

### 2.2 ระบบแม่แบบกิจกรรมรายสัปดาห์ (Weekly Batch)
ฟีเจอร์นี้เหมาะสำหรับกิจกรรมที่มีตารางเวลาซ้ำๆ กันทุกสัปดาห์ เพื่อลดภาระการสร้างใหม่
* **สร้างแม่แบบใหม่:** ไปที่เมนู **ระบบแม่แบบรายสัปดาห์** กรอกข้อมูลกิจกรรมและเลือกวันในสัปดาห์ (เช่น ทุกวันจันทร์ 12:00 น.) แล้วกด **"บันทึกแม่แบบ"**
* **การสร้างลงตารางจริง:** ติ๊กเครื่องหมายถูก (✔️) หน้าแม่แบบที่ต้องการ แล้วกดปุ่ม **"ยืนยันสร้างกิจกรรมที่เลือก"** ระบบจะนำกิจกรรมนั้นไปสร้างลงในสัปดาห์ถัดไปให้อัตโนมัติ
* **การลบแม่แบบ:** สามารถกดปุ่มถังขยะ (🗑️) เพื่อลบแม่แบบที่ไม่ต้องการใช้งานแล้วได้

### 2.3 การแก้ไขและจัดการกิจกรรมที่เปิดอยู่ (Manage Activities)
1. ไปที่เมนู **จัดการกิจกรรม**
2. คลิกที่กิจกรรมที่กำลังเปิดอยู่ (สถานะ OPEN)
3. **สิ่งที่ท่านสามารถจัดการได้:**
   * **✏️ แก้ไข (Edit):** ปรับเปลี่ยนหัวข้อ เปลี่ยนตัวอาจารย์ผู้ดูแล หรือแก้ไขวัน-เวลาเริ่มและจบกิจกรรมได้ (ระบบจะอนุญาตให้แก้ได้หากกิจกรรมยังมีสถานะเป็น OPEN)
   * **นำออก (Remove Attendee):** หากพบผู้ที่ลงชื่อผิดหรือไม่ได้เข้าร่วมกิจกรรมจริง สามารถกดนำชื่อออกได้
   * **ปิดกิจกรรม (Close):** เมื่อสิ้นสุดกิจกรรมแล้ว สามารถกดเพื่อปิดรับการลงชื่อได้ (ระบบป้องกันไม่ให้ปิดก่อนเวลาสิ้นสุดที่ตั้งไว้)
   * **🗑️ ลบกิจกรรม (Delete):** กดเพื่อลบกิจกรรมและข้อมูลการลงชื่อทั้งหมดออกจากระบบ (ใช้ในกรณียกเลิกการจัดกิจกรรม)

## 3. 📊 ระบบติดตามสำหรับ Admin (Dashboards)
Admin สามารถดูรายงานสรุปความก้าวหน้าเพื่อติดตามผลการประเมินได้ที่หน้าจอหลัก:

* **📈 Dashboard ความก้าวหน้า EPA:**
  * ใช้ดูภาพรวมการเก็บเคส EPA และการเข้าร่วมกิจกรรมของแพทย์ทุกคนแบบเรียลไทม์
  * ระบบจะแสดงแถบแจ้งเตือนความเสี่ยง (Risk Alert) สีแดง หากแพทย์ท่านใดมีอัตราสะสมเคสช้ากว่าระยะเวลาที่ผ่านไป
  * สามารถคลิกดูรายละเอียดรายบุคคล และกดปุ่ม **"📥 Export Excel"** เพื่อดาวน์โหลดข้อมูลเป็นไฟล์ .csv นำไปทำรายงานต่อได้
* **🕵️‍♂️ ติดตามสถานะประเมิน 360° (Admin Tracking):**
  * ใช้ตรวจสอบยอดรวมจำนวนใบประเมิน 360 องศาที่แพทย์แต่ละคนได้รับ
  * หากรายชื่อใดแสดงผลเป็น **แถบสีแดง** (ได้ยอดรวมน้อยกว่า 5 ฉบับ) Admin ควรดำเนินการติดตามหรือแจ้งเตือนให้อาจารย์และเจ้าหน้าที่ท่านอื่นช่วยดำเนินการประเมินเพิ่มเติม
