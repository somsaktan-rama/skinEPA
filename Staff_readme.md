⚙️ คู่มือการใช้งานระบบ EPA Skin Tracker

สำหรับ พยาบาลวิชาชีพ, นักวิทยาศาสตร์, Clerk และ เจ้าหน้าที่สายสนับสนุน (Staff / Admin)

ระบบ EPA Skin Tracker ถูกออกแบบมาเพื่ออำนวยความสะดวกให้เจ้าหน้าที่และพยาบาลสามารถร่วมประเมินการทำงานของแพทย์แบบ 360 องศาได้ รวมถึงช่วยให้แอดมินสามารถจัดการกิจกรรมและติดตามความก้าวหน้าของแพทย์ได้อย่างมีประสิทธิภาพ  
1. 🔑 การเข้าสู่ระบบครั้งแรก (Login)

    เปิดแอปพลิเคชันผ่านลิงก์ที่ได้รับ

    กรอก รหัส MDID (รหัส ว. หรือ เบอร์โทรศัพท์ 4-10 หลัก) ในช่องเข้าสู่ระบบ  

    กด เข้าสู่ระบบ ระบบจะผูกบัญชี LINE (LINE Binding) ของท่านอัตโนมัติ  

    การเข้าใช้งานครั้งต่อไป ระบบจะจดจำและเข้าสู่ระบบให้ทันที (Auto-login)  

2. 🔄 การทำแบบประเมิน 360 องศา (สำหรับพยาบาล/เจ้าหน้าที่ทุกคน)

เจ้าหน้าที่ทุกท่านมีส่วนสำคัญในการประเมินทักษะของแพทย์ประจำบ้าน เพื่อนำไปพัฒนาการทำงานร่วมกัน ข้อมูลของท่านจะถูก ปกปิดเป็นความลับ (Anonymous) และนำไปสรุปผลรวมกันเป็นกลุ่ม (เช่น กลุ่มพยาบาล, กลุ่มสายสนับสนุน) เท่านั้น  

    ไปที่เมนู ทำแบบประเมิน 360 องศา

      

    เลือกรายชื่อแพทย์ที่ท่าน เคยร่วมงานด้วยในเทอมนี้

      

    ให้คะแนน (1 = ปรับปรุงมาก ถึง 5 = ดีเยี่ยม) ใน 3 หัวข้อ:  

        การทำงานเป็นทีม (Teamwork)  

        การสื่อสาร (Communication)  

        ความรับผิดชอบ (Punctuality)  

    ระบุข้อเสนอแนะ: ระบบบังคับให้พิมพ์ข้อคิดเห็นตามระดับคะแนน ยิ่งให้คะแนนสูงมาก (5) หรือต่ำมาก (1) จะต้องพิมพ์คำอธิบายที่ละเอียดขึ้น  

    🎙️ วิธีพิมพ์ด้วยเสียง: ระบบไม่อนุญาตให้คัดลอกข้อความมาวาง (Copy/Paste) แนะนำให้ท่านกด ไอคอนไมโครโฟนบนแป้นพิมพ์มือถือ เพื่อพูดแทนการพิมพ์ได้เลยครับ  

3. 📅 การจัดการกิจกรรม (เฉพาะ Staff / Admin)

เจ้าหน้าที่ธุรการหรือ Admin สามารถจัดการการลงชื่อเข้าร่วมกิจกรรมวิชาการได้ดังนี้:  
3.1 การเปิดกิจกรรมใหม่ (แบบครั้งเดียว)

    ไปที่เมนู เปิดกิจกรรมใหม่ (แบบครั้งเดียว)

      

    เลือกรหัสกิจกรรม, พิมพ์หัวข้อ, และระบุอาจารย์ผู้คุม (ถ้ามี)  

    เลือกวัน-เวลาเริ่มต้น และระยะเวลา (เช่น 60 นาที) แล้วกด ยืนยันสร้าง

      

3.2 ระบบแม่แบบกิจกรรมรายสัปดาห์ (Weekly Batch)

เหมาะสำหรับกิจกรรมที่มีตารางเวลาซ้ำๆ กันทุกสัปดาห์  

    สร้างแม่แบบ: ไปที่เมนู ระบบแม่แบบรายสัปดาห์ กรอกข้อมูลกิจกรรมและเลือกวันในสัปดาห์ (เช่น ทุกวันจันทร์ 12:00 น.) แล้วกด "บันทึกแม่แบบ"  

    การสร้างลงตารางจริง: ติ๊กเครื่องหมายถูก (✔️) หน้าแม่แบบที่ต้องการ แล้วกดปุ่ม "ยืนยันสร้างกิจกรรมที่เลือก" ระบบจะนำกิจกรรมนั้นไปกางลงปฏิทินในสัปดาห์ถัดไปให้อัตโนมัติ ป้องกันการสร้างซ้ำ  

3.3 การแก้ไขและปิดกิจกรรม (Manage Activities)

    ไปที่เมนู จัดการกิจกรรม

      

    คลิกที่กิจกรรมที่กำลังเปิดอยู่  

    สิ่งที่ทำได้:

        ✏️ แก้ไข: ปรับเปลี่ยนหัวข้อ เปลี่ยนตัวอาจารย์ หรือแก้เวลาเริ่ม-จบกิจกรรม (ทำได้เมื่อสถานะยังเป็น OPEN)  

        เตะออก: หากพบคนลงชื่อผิด สามารถลบชื่อออกได้  

        ปิดกิจกรรม / 🗑️ ลบกิจกรรม: กดปิดกิจกรรมเมื่อหมดเวลา หรือกดลบหากต้องการยกเลิกกิจกรรมนั้นทิ้ง  

4. 📊 ระบบติดตามสำหรับ Admin (Dashboards)

Admin สามารถดูรายงานสรุปความก้าวหน้าเพื่อติดตามผลได้ที่หน้าจอหลัก:  

    📈 Dashboard ความก้าวหน้า EPA:
    ดูภาพรวมการเก็บเคส EPA และ Activity ของแพทย์ทุกคน ระบบจะแสดงแถบเตือนความเสี่ยง (Risk Alert) สีแดง หากแพทย์ท่านใดเก็บเคสล่าช้ากว่าเวลาที่ผ่านไป สามารถกดปุ่ม "📥 Export Excel" เพื่อโหลดข้อมูลไปทำรายงานต่อได้  

    🕵️‍♂️ ติดตามสถานะ 360 (Admin):
    ใช้เช็คว่าแพทย์แต่ละคนได้รับใบประเมินครบ 5 ฉบับแล้วหรือไม่ หากรายชื่อใดเป็น แถบสีแดง Admin สามารถแจ้งเตือนให้เจ้าหน้าที่หรืออาจารย์ท่านอื่นช่วยประเมินเพิ่มเติมได้  

🔧 คู่มือการใช้งานระบบ EPA Skin Tracker

สำหรับผู้ดูแลระบบ (Superstaff)

สิทธิ์ Superstaff ถูกออกแบบมาเพื่อจัดการฐานข้อมูลบุคลากร และแก้ไขปัญหาการผูกบัญชี LINE ให้กับผู้ใช้งานท่านอื่นผ่านหน้าเว็บได้โดยตรง โดยไม่ต้องเข้าไปแก้ไขใน Google Sheets  
1. การเข้าสู่หน้า Superstaff

    ล็อกอินเข้าสู่ระบบด้วยรหัสที่มีสิทธิ์ Superstaff  

    ที่หน้าเมนูหลัก จะพบปุ่มกรอบสีแดง "🔧 จัดการข้อมูลบุคลากร" อยู่ด้านล่างสุด  

2. การค้นหาและแก้ไขข้อมูลบุคลากร

    ในหน้า Superstaff ให้กรอก MDID (4-10 หลัก) ของบุคคลที่ต้องการแก้ไขในช่องค้นหา แล้วกด ดึงข้อมูล

      

    ระบบจะแสดงข้อมูลทั้งหมดของบุคลากรท่านนั้น พร้อมภาพโปรไฟล์ และเวลาที่ล็อกอินเข้าใช้งานล่าสุด (Last Login)  

    ท่านสามารถแก้ไขฟิลด์ข้อมูลได้ทันที เช่น ชื่อ, ชั้นปี (R2/R3/R4), สถานะ, Email และ Mobile  

    เมื่อแก้ไขเสร็จ ให้เลื่อนลงมากดปุ่ม "บันทึกข้อมูล" สีเขียว  

3. การจัดการสิทธิ์พิเศษสำหรับอาจารย์

ในหน้าแก้ไขข้อมูลบุคลากร จะมี 2 ฟิลด์สำคัญสำหรับการจัดการสิทธิ์ของ Instructor:  

    Eval Target: ระบุ MDID ของลูกศิษย์ (คั่นด้วยเครื่องหมายจุลภาค ,) เพื่อกำหนดเป้าหมายว่าอาจารย์ท่านนี้ต้องประเมินใครบ้างเป็นพิเศษ  

    View Target: กำหนดว่าอาจารย์ท่านนี้สามารถดูผลประเมิน 360 องศาของลูกศิษย์คนไหนได้บ้าง (หากปล่อยว่างไว้ ปุ่มเมนู "ดูผลประเมิน 360" จะไม่แสดงบนหน้าจอของอาจารย์ท่านนั้น)  

4. 🔓 การรีเซ็ตการผูกบัญชี LINE (Unbind LINE)

หากมีผู้ใช้งานแจ้งว่า "ล็อกอินไม่ได้ เนื่องจากบัญชีถูกผูกกับเครื่องอื่นไปแล้ว" Superstaff สามารถปลดล็อกให้ได้ทันทีดังนี้:  

    ค้นหา MDID ของผู้ที่พบปัญหา  

    เลื่อนหาช่อง LINE UID (ลบออกเพื่อรีเซ็ตเครื่องใหม่ได้)

      

    ลบข้อความรหัสภาษาอังกฤษยาวๆ ในช่องนั้นทิ้งให้ว่างเปล่า

      

    กดปุ่ม บันทึกข้อมูล

      

    แจ้งให้ผู้ใช้งานกลับไปล็อกอินที่มือถือเครื่องใหม่ ระบบจะทำการผูกบัญชีกับ LINE เครื่องใหม่ให้อัตโนมัติครับ
