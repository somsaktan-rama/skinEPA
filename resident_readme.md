# 📖 คู่มือการใช้งานระบบ EPA Skin Tracker
**สำหรับแพทย์ประจำบ้าน (Resident) และแพทย์ต่อยอด (Fellow)**

ยินดีต้อนรับสู่ระบบ EPA Skin Tracker v2.0 ระบบนี้ถูกออกแบบมาเพื่อช่วยให้แพทย์ประจำบ้านและแพทย์ต่อยอดสามารถติดตามความก้าวหน้าของการประเมิน EPA, ลงชื่อเข้าร่วมกิจกรรม และทำแบบประเมิน 360 องศา ได้อย่างสะดวกผ่าน LINE

---

## 1. 🔑 การเข้าสู่ระบบครั้งแรก (Login & LINE Binding)
เพื่อความสะดวกในการใช้งานระยะยาว ระบบจะทำการผูกบัญชีผู้ใช้เข้ากับ LINE ของท่าน[cite: 3]
1. เปิดแอปพลิเคชันผ่านลิงก์ใน LINE
2. กรอก **รหัส MDID** (รหัส ว. หรือ เบอร์มือถือ 4-10 หลัก) ในช่องเข้าสู่ระบบ[cite: 4]
3. กด **เข้าสู่ระบบ** ระบบจะทำการผูกบัญชี LINE (LINE Binding) อัตโนมัติในครั้งแรก[cite: 3]
4. ในครั้งต่อไป ระบบจะจดจำท่านและทำการเข้าสู่ระบบอัตโนมัติ (Auto-login) ทันที[cite: 3]

> **💡 หมายเหตุ:** หากท่านเปลี่ยนโทรศัพท์มือถือ หรือต้องการย้ายเครื่องเข้าใช้งาน สามารถกดปุ่ม **"ต้องการรีเซ็ตการผูกบัญชี LINE?"** ระบบจะส่งลิงก์ยืนยันไปที่อีเมลของท่าน[cite: 3, 4]

## 2. 📈 ความก้าวหน้าของฉัน (My Progress Dashboard)
เมนูนี้ช่วยให้ท่านติดตามภาพรวมการศึกษาและจำนวนเคสที่ต้องเก็บตลอดปีการศึกษา[cite: 4]
* **ความสำเร็จรวมทั้งหมด:** แสดงเปอร์เซ็นต์ (%) ความก้าวหน้าเปรียบเทียบกับเป้าหมายที่กำหนดไว้ในชั้นปีของท่าน[cite: 3]
* **แถบสีสถานะ:**
  * 🟩 **สีเขียว:** เก็บเคสครบตามเป้าหมายแล้ว[cite: 4]
  * 🟧 **สีส้ม:** เก็บเคสได้เกิน 50% แต่ยังไม่ครบเป้าหมาย[cite: 4]
  * 🟥 **สีแดง:** เก็บเคสได้น้อยกว่า 50%[cite: 4]
* **⚠️ แจ้งเตือนความเสี่ยง (Risk Alert):** หากเปอร์เซ็นต์การเก็บเคสของท่านล่าช้ากว่าระยะเวลาปีการศึกษาที่ผ่านไป ระบบจะแสดงแถบแจ้งเตือนสีแดง เพื่อให้ท่านเร่งเก็บเคสเพิ่มเติม[cite: 3, 4]

## 3. 📝 ระบบประเมิน EPA
การส่งแบบฟอร์มประเมิน EPA ทำได้ง่ายๆ ผ่านสมาร์ทโฟน[cite: 4]
1. ไปที่เมนู **สร้างฟอร์ม EPA**[cite: 4]
2. เลือกรายการ EPA ที่ต้องการประเมิน[cite: 4]
3. เลือก **อาจารย์ผู้คุม** (ระบบจะแสดงเฉพาะรายชื่ออาจารย์ที่มีโควตาประเมินว่าง)[cite: 4]
4. **ระบุ Note (จำเป็น):** พิมพ์บันทึกรายละเอียดหัตถการหรือความเห็นของท่าน[cite: 4]
5. กด **บันทึกส่ง** (ระบบจะส่ง LINE แจ้งเตือนตรงไปยังอาจารย์ผู้คุมทันที)[cite: 3, 4]

### 📜 การตรวจสอบประวัติ (History)
ท่านสามารถดูประวัติการส่ง EPA ย้อนหลังได้ที่เมนู **ดูประวัติการส่ง EPA** เพื่อตรวจสอบสถานะ:
* ⏳ **รอ:** อาจารย์ยังไม่ได้ทำการประเมิน[cite: 4]
* ✅ **PASS:** ผ่านเกณฑ์ประเมิน (แสดงผลคะแนนและ Feedback จากอาจารย์)[cite: 4]
* ❌ **FAIL:** ไม่ผ่านเกณฑ์ประเมิน (แสดงเหตุผลและข้อเสนอแนะเพื่อนำไปปรับปรุง)[cite: 4]

## 4. 📍 การลงชื่อเข้าร่วมกิจกรรม (Activity Check-in)
ระบบ Check-in แบบเรียลไทม์ สำหรับกิจกรรมวิชาการและการเรียนการสอน[cite: 4]
1. ไปที่เมนู **ลงชื่อเข้าร่วมกิจกรรม**[cite: 4]
2. ระบบจะแสดงรายการกิจกรรมที่กำลังเปิดอยู่[cite: 4]
3. ท่านสามารถกดปุ่ม **"ลงชื่อ (Join)"** เพื่อบันทึกเวลาเข้าร่วมได้[cite: 4]

> **⏱️ เงื่อนไขเวลา:** ปุ่มลงชื่อจะเปิดให้กดได้ตั้งแต่ **30 นาทีก่อนเริ่มกิจกรรม** ไปจนถึงเวลาสิ้นสุดกิจกรรมเท่านั้น หากหมดเวลาจะไม่สามารถลงชื่อได้[cite: 3]

## 5. 🔄 ระบบประเมิน 360 องศา (360° Evaluation)
ระบบสำหรับประเมินเพื่อนร่วมงาน เพื่อการพัฒนาทักษะแบบรอบด้าน[cite: 4]

### ✍️ การทำแบบประเมินให้เพื่อนแพทย์
1. ไปที่เมนู **ทำแบบประเมิน 360 องศา**[cite: 4]
2. เลือกรายชื่อแพทย์ที่ท่าน **"รู้จักและเคยร่วมงานด้วยในเทอมนี้"**[cite: 4]
3. ให้คะแนน (1 = ปรับปรุงมาก ถึง 5 = ดีเยี่ยม) ใน 3 หัวข้อ:[cite: 4]
   * การทำงานเป็นทีม (Teamwork)[cite: 4]
   * การสื่อสาร (Communication)[cite: 4]
   * ความรับผิดชอบ/ตรงต่อเวลา (Punctuality)[cite: 4]
4. **ระบุข้อเสนอแนะ:** เพื่อการประเมินที่มีคุณภาพ ระบบบังคับให้พิมพ์ข้อคิดเห็นตามระดับคะแนนที่ให้ (คะแนน 1 และ 5 ต้องเขียนอธิบายเยอะกว่าคะแนนระดับกลาง)[cite: 4]
5. **🎙️ Speech-to-Text:** ระบบไม่อนุญาตให้ Copy/Paste ข้อความ แนะนำให้ใช้ **ปุ่มไมโครโฟนบนแป้นพิมพ์มือถือ** เพื่อพูดแทนการพิมพ์[cite: 4]

### 📊 การดูผลตอบรับของตนเอง (My 360 Feedback)
ท่านสามารถเข้ามาดูผลประเมินที่คนอื่นประเมินท่านได้ที่เมนู **ดูผลตอบรับ 360**[cite: 4]
* ข้อมูลทั้งหมดจะถูก **ปกปิดตัวตน (Anonymous)**[cite: 4]
* สรุปคะแนนเฉลี่ยและข้อคิดเห็น จะถูกแยกตามกลุ่มผู้ประเมิน (อาจารย์, เพื่อนแพทย์, พยาบาล, สายสนับสนุน) และสรุปแยกเป็นรายเทอม[cite: 3, 4]
